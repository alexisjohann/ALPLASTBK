### 4. Zentrale Proposition
**Proposition (Non-Concavity from Complementarity):**
> If ∂²Π/∂x_i∂x_j > 0 for all i ≠ j, then Π is not globally concave.
> The payoff landscape has multiple local maxima.

**Beweis via Hessian:** ✅
- Diagonal H_ii < 0 (own concavity)
- Off-diagonal H_ij > 0 (complementarity)
- → At least one positive eigenvalue
- → Not negative semi-definite
- → Not globally concave
