## Inhalt

Die kontextabhängige Referenzstruktur:

1. **Was ist C*?**: Das Optimum im Kontext Ψ
2. **Adaptive Referenzpunkte**: Nicht statisch, sondern dynamisch
3. **Misfit als Signal**: Abweichung C - C* als Information
4. **Lernarchitektur**: Wie C* aktualisiert wird

```

      ↓

      ↓
Iteration schließt den Kreis
```
