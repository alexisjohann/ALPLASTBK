## Inhalt

Praktische Implikationen für Policy-Maker:

1. **Intervention Design**: Wie Kontext berücksichtigen?
2. **Timing**: Wann eingreifen? (Moment of Truth)
3. **Targeting**: Wer profitiert? (Heterogenität)
4. **Unintended Consequences**: Was kann schiefgehen?
5. **Evaluation**: Wie Erfolg messen?

---

## Schlüsselkonzepte

| Konzept | Implikation |
|---------|-------------|
| **Komplementarität** | Partielle Interventionen können scheitern |
| **Kontext-Abhängigkeit** | Eine Größe passt nicht für alle |
| **Awareness** | Menschen müssen Nutzen *sehen* |
| **Willingness** | Sehen reicht nicht — Barrieren abbauen |

## Policy-Prinzipien aus 10C
