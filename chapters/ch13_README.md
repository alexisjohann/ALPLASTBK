## Inhalt

Die Behavioral Change Journey beschreibt den Prozess der Verhaltensänderung:

1. **Stages**: Unaware → Aware → Considering → Intending → Acting → Maintaining
2. **Intervention Timing**: Welche Intervention in welcher Phase wirkt
3. **Stage Factors**: $
\alpha_{BCJ}$ als stufenabhängiger Multiplikator
4. **Intention-Action Gap**: Übergang von WAX zu tatsächlicher Handlung
