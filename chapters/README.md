## 📝 Wie man ein Kapitel schreibt

### 1. Template verwenden

```bash
cp 00_chapter_template.tex [NN]_[name].tex
```

### 2. Struktur einhalten

```
OPENING (Pflicht)
├── \section{} Heading mit Label
├── Opening Paragraph (WAS und WARUM)
└── Chapter Roadmap

CORE CONTENT
├── \subsection{} für Hauptthemen
├── \paragraph{} für Schlüsselpunkte
├── Pattern: Intuition → Beispiel → Formal → Interpretation
└── Kerngleichungen (nummeriert, interpretiert)

CLOSING (Pflicht)
├── Summary
├── Appendix-Verweise
└── Überleitung zum nächsten Kapitel
```

### 3. Appendix-Verbindung herstellen

Jedes Konzept-Kapitel sollte auf den entsprechenden Appendix verweisen:

```latex
Die vollständige Axiomatisierung findet sich in Appendix~B (CORE-HOW).
```
