# Tiefenprüfung Protokoll v46

## Prüfkategorien (7 Dimensionen)

### A. INHALTLICHE KOHÄRENZ
- [ ] Logischer Aufbau innerhalb des Kapitels
- [ ] Roter Faden erkennbar
- [ ] Keine Widersprüche
- [ ] Angemessene Tiefe für Zielgruppe

### B. CLAIM-EVIDENCE MATCHING
- [ ] Jede empirische Behauptung belegt
- [ ] Keine unbelegten Kausalaussagen
- [ ] Evidenzstärke angemessen kommuniziert
- [ ] Keine Übertreibungen

### C. BEISPIEL-VALIDIERUNG
- [ ] Zahlen in Beispielen plausibel
- [ ] Beispiele illustrieren Konzept korrekt
- [ ] Keine rechnerischen Fehler
- [ ] Einheiten konsistent

### D. KAPITEL-ÜBERGÄNGE
- [ ] Verbindung zum vorherigen Kapitel
- [ ] Vorbereitung auf nächstes Kapitel
- [ ] Keine abrupten Themenwechsel
- [ ] Zusammenfassung am Ende

### E. TERMINOLOGIE
- [ ] Fachbegriffe bei Erstverwendung definiert
- [ ] Konsistente Verwendung im Kapitel
- [ ] Konsistent mit anderen Kapiteln
- [ ] Glossar-Referenz wo nötig

### F. FRAMEWORK-INTEGRATION
- [ ] Konzepte korrekt im Framework verortet
- [ ] Notation korrekt angewendet
- [ ] Verbindung zu C*, K, Q klar
- [ ] Ψ-Dimensionen korrekt referenziert

### G. WISSENSCHAFTLICHE STANDARDS
- [ ] Keine logischen Fehlschlüsse
- [ ] Angemessene Einschränkungen (caveats)
- [ ] Keine versteckten Annahmen
- [ ] Falsifizierbarkeit gegeben
