# DIE VERZAUBERUNG
## Location Scouting Brief

**Kriterienkatalog für Drehorte**

---

## EXECUTIVE SUMMARY

| Parameter | Anforderung |
|-----------|-------------|
| **Hauptdrehort** | Alpines Bergdorf, 800-1200m Höhe |
| **Epoche** | 1930er Jahre authentisch oder rekonstruierbar |
| **Region** | Österreich (Tirol, Salzburg), Schweiz (Graubünden), Südtirol |
| **Drehtage** | ca. 45-55 Tage |
| **Jahreszeit** | Spätsommer (Aug-Sept) + Winter (Dez-Jan) |
| **Infrastruktur** | Basis-Catering, Unterkunft für 50+ Crew in 30km Radius |

---

## DORF-ANFORDERUNGEN (HAUPTDREHORT)

### Must-Have

```
☑ Geschlossene Dorfstruktur (keine modernen Einsprengsel)
☑ Kirche mit Turm (zentral, dominant)
☑ Gasthof/Wirtshaus (authentisch, Innenräume nutzbar)
☑ Mehrere historische Bauernhöfe
☑ Keine sichtbaren Stromleitungen, Satellitenschüsseln
☑ Bergpanorama sichtbar
☑ Straße/Pfad, der "von außen" ins Dorf führt
☑ Abgeschiedenheit vermittelbar (keine Autobahn in Sichtweite)
```

### Nice-to-Have

```
○ Platz für Crew-Base in der Nähe
○ Bestehendes Dorf (nicht Freilichtmuseum)
○ Kooperative Gemeinde
○ Bereits für Filme genutzt
○ Förderung durch Region
```

### Dealbreaker

```
✗ Moderne Neubauten im Zentrum
✗ Sichtbare Skipisten/Lifte
✗ Durchgangsstraße mit Verkehr
✗ Touristische Überprägung (viele Hotels, Souvenirläden)
✗ Flaches Gelände ohne Bergkulisse
```

## JAHRESZEITEN-PLAN

### Phase 1: Spätsommer (August-September)

```
SZENEN: Akt I + Akt II (bis Ritual)
LICHT: Warm, golden
VEGETATION: Ernte, reif
DAUER: 25-30 Drehtage
```

| Monat | Szenen | Locations |
|-------|--------|-----------|
| August | 1-16 | Dorf, Felder, Gasthaus |
| September | 17-28 | Waldlichtung, Dorf, Hof |

### Phase 2: Drehpause (Oktober-November)

```
ZWECK: Vegetationswandel, Vorbereitung Winterdreh
AKTIVITÄTEN: Post-Production Phase 1, Schneeplanung
```

### Phase 3: Winter (Dezember-Januar)

```
SZENEN: Akt III (ab Szene 38)
LICHT: Kalt, grau, weiß
VEGETATION: Schnee
DAUER: 15-20 Drehtage
```

| Monat | Szenen | Locations |
|-------|--------|-----------|
| Dezember | 37-41 | Bergplateau, Dorf im Schnee |
| Januar | 42-44 | Arzthaus, Gisson-Hof |

## BUDGET-RELEVANTE FAKTOREN

| Faktor | Auswirkung |
|--------|------------|
| Schweiz vs. Österreich | CH ca. 40% teurer |
| Touristensaison | Hochsaison +50% Unterkunft |
| Winterdreh | Heizung, Sicherheit, Kürzere Drehtage |
| Entfernung zur Stadt | Transport, Unterbringung Crew |
| Denkmalschutz | Einschränkungen, aber oft Förderung |
| Bewohnte vs. Museum | Bewohnt: Koordination nötig |
