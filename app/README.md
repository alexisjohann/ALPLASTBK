# UBS Choice Architecture Simulator

Interactive UI for behavioral intervention design based on the EBF 10C Framework.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run choice_architecture_ui.py
```

## Features

- **Context Sliders (Ψ):** Adjust Trust, Digital Adoption, Risk Aversion, Income Stability
- **Real-time Predictions:** Monte Carlo simulation with 90% confidence intervals
- **Intervention Selection:** Choose interventions and see combined effects
- **Segment Comparison:** Compare predictions across customer segments

## Model

- **ID:** UBS-FIN-SB-001
- **Framework:** Evidence-Based Framework (EBF) - 10C Methodology
- **Status:** CONFIG-derived, Validation Pending
