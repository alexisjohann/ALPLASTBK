Title: Minimizing a Submodular Function on a Lattice
Authors: Donald M. Topkis
Year: 1978
Journal: Operations Research, Vol. 26, No. 2, pp. 305-321

================================================================================
ABSTRACT
================================================================================

This paper gives general conditions under which a collection of optimization
problems, with the objective function and the constraint set depending on a
parameter, has optimal solutions that are an isotone (monotonically increasing)
function of the parameter.

The central contribution is a theory exploring the problem of minimizing a
submodular function on a lattice. A function is submodular if it exhibits
decreasing differences: the marginal value of increasing one variable decreases
when other variables increase. Conversely, supermodular functions exhibit
increasing differences (complementarity).

The paper establishes what became known as "Topkis's Theorem": if the objective
function is supermodular in (x, θ) and the constraint set is a lattice, then
the argmax is nondecreasing in the parameter θ.

================================================================================
KEY FINDINGS
================================================================================

1. TOPKIS'S THEOREM: For supermodular objectives on lattices, optimal solutions
   are monotone in parameters - foundational for comparative statics

2. SUBMODULARITY/SUPERMODULARITY: Formal characterization of decreasing vs
   increasing differences between choice variables

3. LATTICE STRUCTURE: When constraint sets form lattices, powerful monotonicity
   results become available

4. COMPARATIVE STATICS: Results enable comparative statics analysis without
   requiring differentiability of objective functions

5. COMPLEMENTARITY FORMALIZATION: Provides mathematical foundation for
   analyzing complementarities in economics

6. OPTIMIZATION ALGORITHMS: Submodularity enables efficient algorithms for
   otherwise difficult optimization problems
