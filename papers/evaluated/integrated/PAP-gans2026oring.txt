================================================================================
ABSTRACT
================================================================================

This paper studies AUTOMATION when tasks are QUALITY COMPLEMENTS rather than
separable. Production requires numerous tasks whose qualities MULTIPLY as in
an O-RING TECHNOLOGY.

THE MODEL:
- Worker allocates fixed time endowment across tasks
- Machines can replace tasks with given quality
- Time is allocated across remaining manual tasks

THE "FOCUS" MECHANISM generates THREE KEY RESULTS:

1. TASK-BY-TASK SUBSTITUTION LOGIC IS INCOMPLETE
   Automating one task changes the return to automating others

2. AUTOMATION DECISIONS ARE DISCRETE
   Can require BUNDLED ADOPTION even when automation quality improves smoothly

3. LABOUR INCOME CAN RISE UNDER PARTIAL AUTOMATION
   Automation SCALES the value of remaining bottleneck tasks

CRITICAL POLICY IMPLICATION:
Widely-used EXPOSURE INDICES (which aggregate task-level automation risk
using LINEAR formulas) will OVERSTATE DISPLACEMENT when tasks are complements.

The relevant object is NOT average task exposure but the STRUCTURE OF
BOTTLENECKS and how automation reshapes worker time around them.

================================================================================
KEY FINDINGS
================================================================================

1. O-ring = multiplicative task complementarity
2. Linear exposure indices OVERSTATE displacement
3. Task-by-task automation logic is incomplete
4. Bundled adoption may be required
5. Labour income can RISE with partial automation
6. Bottleneck structure matters, not average exposure
