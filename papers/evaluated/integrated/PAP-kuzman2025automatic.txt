================================================================================
ABSTRACT
================================================================================

COMPREHENSIVE SURVEY of Automatic Genre Identification (AGI), a text
classification task focused on genres.

DEFINITION OF GENRE:
Text categories defined by:
- Author's PURPOSE
- Common FUNCTION of the text
- Text's CONVENTIONAL FORM

SURVEY COVERAGE:

1. GENRE DEFINITIONS AND TERMINOLOGY
   - Genre vs. Register vs. Text Type
   - Conceptual distinctions

2. GENRE SCHEMATA COMPARISON
   - 16 different schemata analyzed
   - 280 genre labels across systems

3. DATASET ANALYSIS
   - CORE: 48k documents
   - Leeds: 5k documents
   - GINCO: 1k documents
   - FTD and others

4. MACHINE LEARNING METHODS
   - From SVM to Transformers
   - Evolution of approaches

KEY FINDING (Performance):
- XLM-RoBERTa achieves F1 0.72-0.84 on multilingual CORE
- Zero-shot transfer: F1 0.61-0.69

RELEVANCE FOR EBF:
Highly relevant for Scientific Writing Style Module (SWSM):
- E7: 8D → Genre Mapper
- E8: Move Planner components
