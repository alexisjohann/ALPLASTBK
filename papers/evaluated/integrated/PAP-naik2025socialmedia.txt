Title: Impact of Social Media on Cognitive Development of Children and Young Adults: A Systematic Review
Authors: Vaishnavi Subhash Naik, Edlin Glane Mathias, Priyanka Krishnan, Vanitha Jagannath
Year: 2025
Journal: BMC Pediatrics, Vol. 25, 826
DOI: 10.1186/s12887-025-06041-5

================================================================================
ABSTRACT
================================================================================

This SYSTEMATIC REVIEW evaluates the impact of social media usage on
cognitive development in children and young adults (ages 9-25).

METHODOLOGY: Following PRISMA 2020 guidelines
- 23 studies included
- Total sample: n = 42,380 participants

COGNITIVE DOMAINS EXAMINED:
1. ATTENTION
2. MEMORY (working memory)
3. EXECUTIVE FUNCTION
4. LANGUAGE DEVELOPMENT

KEY FINDINGS - MIXED EFFECTS:

NEGATIVE ASSOCIATIONS:
- Excessive use → impaired attention
- Heavy use → reduced working memory capacity
- Overuse → diminished executive functioning

POSITIVE ASSOCIATIONS:
- Certain platforms showed benefits for LANGUAGE SKILLS
- Educational engagement through social media can enhance learning
- Interactive features may support cognitive development

MODERATORS:
- Type of platform matters
- Duration and frequency of use
- Educational vs. entertainment content
- Age of user

================================================================================
KEY FINDINGS
================================================================================

1. Systematic review: 23 studies, n=42,380
2. MIXED EFFECTS on cognitive development
3. Excessive use: negative for attention, memory, executive function
4. Some platforms: positive for language skills
5. Educational engagement can be beneficial
6. PRISMA 2020 methodology
