## Appendix: Behavioral Economics Foundation

### Why These Segments Behave Differently

**Loss Aversion (λ) Explains Channel Preference:**

```
Seg 1: λ=2.8 → Fears digital mistakes → Needs human validation
Seg 2: λ=1.8 → Risk-tolerant → Embraces digital
Seg 4: λ=2.5 → Relationship = insurance → Omnichannel safety
```

**Present Bias (β) Explains Research Behavior:**

```
Seg 3: β=0.88 → Patient → Research-intensive
Seg 2: β=0.75 → Moderate bias → Quick execution
Seg 5: β=0.80 → Research ahead → Decide later (time inconsistency)
```

**Trust (trust_inst) Explains Loyalty Drivers:**

```
Seg 1: trust=0.85 → Institution > product
Seg 4: trust=0.80 → Journey consistency = trust
Seg 2: trust=0.65 → Platform performance > brand
```

**Key Insight from MS-CJ-002:**
- **NOT** about channel preference (online vs. offline)
- **ABOUT** how journey quality maps to underlying behavioral parameters
- **Segment 4** high trust + high λ → journey breaks DESTROY value
