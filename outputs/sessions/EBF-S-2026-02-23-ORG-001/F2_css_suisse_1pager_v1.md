# CSS - Center for Species Survival Suisse

## Das Problem

**35% aller untersuchten Arten in der Schweiz sind gefaehrdet** -- deutlich mehr als im globalen Durchschnitt (28%). Bei Amphibien sind es 79%, bei Voegeln 40%. 233 Arten sind bereits ausgestorben, 506 weitere stehen unmittelbar davor. Das Fachwissen fuer den Artenschutz ist vorhanden -- aber ueber Dutzende Institutionen fragmentiert.

## Die Loesung: CSS Suisse

Ein **nationales Kompetenzzentrum**, das bestehendes Artenschutz-Wissen buendelt, koordiniert und fuer konkrete Schutzprogramme verfuegbar macht.

| Funktion | Beschreibung |
|----------|-------------|
| **Wissen buendeln** | Zentrale Plattform fuer Artenschutz-Expertise (Tiere, Pflanzen, Lebensraeume) |
| **Handeln koordinieren** | Erhaltungszucht, Schutzprogramme und Wiederansiedlungen abstimmen |
| **Behoerden beraten** | BAFU, Kantone und internationale Gremien wissenschaftlich unterstuetzen |
| **Sensibilisieren** | Bevoelkerung ueber Mitgliederzoos und Bildungsprogramme erreichen |

## Internationales Vorbild

Das CSS-Modell basiert auf dem **IUCN Center for Species Survival Netzwerk** (Ziel: 1 CSS pro Kontinent) und ist inspiriert durch **Irlands National Biodiversity Action Plan** (2023-2030, gesetzlich verankert). Die Schweiz als IUCN-Sitzland hat eine besondere Verantwortung.
