## 2.2 Neues Modell: PMWT-1 (Pharma-MFN Welfare Transfer)

### Theoretische Basis

| Theorie | Autor | Jahr | Kern-Einsicht | EBF-Mapping |
|---------|-------|------|---------------|-------------|
| **Public Choice** | Olson | 1965 | Concentrated benefits, diffuse costs | WHO, HIERARCHY |
| **Regulatory Capture** | Stigler | 1971 | Regulierung dient den Regulierten | Ψ_I |
| **Rent-Seeking** | Tullock | 1980 | Ressourcen → Lobbying statt Produktion | WHAT |

### 10C-Mapping

| CORE | Frage | Anwendung im Modell |
|------|-------|---------------------|
| **WHO** | Wer hat Utility? | Prämienzahler (-), Pharma-Aktionäre (+), Fiskus (+) |
| **WHAT** | Was ist Utility? | Finanzielle Wohlfahrt (CHF) |
| **HOW** | Wie interagieren? | Nullsumme zwischen Gruppen |
| **WHEN** | Wann zählt Kontext? | MFN-Policy als exogener Schock |
| **HIERARCHY** | Stratifizierung? | 8.8 Mio. vs. ~10 Konzerne |

### Stakeholder-Struktur

| Stakeholder | N | Effekt bei ΔP↑ | Organisationsgrad |
|-------------|---|----------------|-------------------|
| CH-Prämienzahler | 8.8 Mio. | **Negativ** | Gering |
| Pharma-Aktionäre (Ausland) | ~60% | **Positiv** | Sehr hoch |
| Pharma-Aktionäre (CH) | ~40% | **Positiv** | Sehr hoch |
| Pharma-Mitarbeiter | 45'000 | Neutral | Mittel |
| US-Patienten | 330 Mio. | Negativ (indirekt) | Irrelevant für CH |

### Wohlfahrtsformeln

**Verlust Prämienzahler:**
```
ΔW_Prämienzahler = -ΔP_CH × Q_CH_patent
```

**Gewinn Pharma:**
```
ΔW_Pharma = +ΔP_CH × Q_CH_patent × (1 + ε × Q_US/Q_CH)
```

**Netto-Wohlfahrt Schweiz:**
```
Netto_CH = ΔW_Prämienzahler + (1 - τ_Ausland) × ΔW_Pharma_CH + Steuereffekt
```

Wobei:
- `ΔP_CH` = Preisänderung in Prozent
- `Q_CH_patent` = Patentgeschützte Medikamente (~4.9 Mrd. CHF)
- `ε` = Spillover-Elastizität CH → US (~0.12)
- `τ_Ausland` = Anteil ausländischer Aktionäre (~63%)

E[Mehrkosten] = Σ P(Si) × Kosten(Si)

## 4.5 Validierung «Standort-Argument»

### Behauptete Kausalkette (Interpharma)

```
P_CH ↑ → Profite ↑ → Jobs_CH ↑ → Wohlfahrt_CH ↑
```

### Tatsächliche Kausalkette (Evidenz)

```
P_CH ↑ → Profite ↑ → 63% fliessen ans Ausland
                   → 37% bleiben in CH
                   → Jobs_CH ≈ konstant (nicht preisabhängig)
```

### Faktencheck

| Prüfpunkt | Behauptung | Realität |
|-----------|------------|----------|
| CH-Markt global | Wichtig | **2-3%** des Weltumsatzes |
| Preisabhängigkeit Jobs | Hoch | **Keine Evidenz** |
| Standortfaktoren | Preise | **Talent, F&E-Cluster, Stabilität** |
| Abwanderungsrisiko | Real | **Unglaubwürdig** (wegen 2-3%) |

### Ökonomische Einordnung

Das «Standort-Argument» ist klassisches **Rent-Seeking** (Tullock 1980):
- **Konzentrierte Vorteile:** ~10 Pharma-Konzerne gewinnen
- **Diffuse Kosten:** 8.8 Mio. Prämienzahler zahlen
- **Politisches Framing:** «Standort» verschleiert Transfer
- **Ressourcenverschwendung:** Lobbying-Kosten statt Produktion
