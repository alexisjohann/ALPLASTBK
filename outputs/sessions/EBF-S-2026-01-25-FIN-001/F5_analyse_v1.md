## 2. Bewertungsmethoden

### 2.1 Ertragswertmethode (MFH-Perspektive)

**Annahmen:**
- Durchschnittliche Wohnungsgrösse: ~85 m²
- Mietpreis Kesswil Seelage: CHF 18-22/m²/Monat
- Brutto-Jahresmiete: CHF 56'000 - 68'500
- Bewirtschaftungskosten: 15%
- Unterhalt Altbau: 10%
- Netto-Jahresmiete: CHF 42'000 - 51'400
- Kapitalisierungssatz: 3.75% - 4.25%

**Ergebnis:**
```
Ertragswert = CHF 42'000 - 51'400 × (1 / 0.04)
            = CHF 1'050'000 - 1'285'000
            ≈ CHF 1'100'000 - 1'300'000
```

### 2.2 Marktwertmethode (EFH-Perspektive)

**Annahmen:**
- Durchschnittspreis Thurgau: CHF 7'300/m²
- Seelage-Premium (2. Reihe): +25%
- Geschätzter m²-Preis Kesswil Seelage: CHF 9'000 - 9'500

**Ergebnis:**
```
Marktwert = 260 m² × CHF 9'000 - 9'500
          = CHF 2'340'000 - 2'470'000
          ≈ CHF 2'400'000
```

### 2.3 Hybrid-Methode (Empfohlen)

**Flexibilitätsprämie:**

Das Objekt kann OHNE Umbau sowohl als EFH als auch als MFH genutzt werden. Diese Flexibilität erweitert den Käuferkreis (Investoren + Eigennutzer) und rechtfertigt eine Prämie.

```
Flexibilitätsprämie = α × (V_Markt - V_Ertrag)
                    = 0.35 × (CHF 2'400'000 - CHF 1'200'000)
                    = CHF 420'000 - 600'000

Hybrid-Wert = V_Ertrag + Flexibilitätsprämie
            = CHF 1'200'000 + CHF 500'000 - 700'000
            = CHF 1'700'000 - 1'900'000
```
