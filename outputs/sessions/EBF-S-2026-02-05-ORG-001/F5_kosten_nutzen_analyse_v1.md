## Executive Summary

Die Implementierung des **SAP SuccessFactors Job Profile Builder (JPB)** wird für Migros Luzern **empfohlen**. Das Modul ist Teil des Talent Management Bundles (keine separate Lizenz), die Migros Gruppe nutzt bereits SuccessFactors, und Schweizer Retail-Referenzen (Coop, ALDI) bestätigen die Eignung.

**Geschätzte Investition:** CHF 20'000-35'000
**Break-even:** 18-24 Monate

## 2. Was ist der Job Profile Builder?

Der Job Profile Builder (JPB) ist ein Plattform-Tool in SAP SuccessFactors:

- **Zentrale Verwaltung** von Stellenprofilen (Verantwortlichkeiten, Skills, Kompetenzen, Qualifikationen)
- **Single Source of Truth** für alle HR-Module
- **Basiert auf MDF** (Meta Data Framework)
- **Ersetzt** die Legacy-Funktion "Families & Roles"

### Modul-Integrationen
- Recruiting (Job Descriptions)
- Performance (Performance Forms, 360 Reviews)
- Career Development (Career Worksheet, Career Plan)
- Talent Intelligence Hub (AI-Skills Framework)
- Learning Management Solution

## 3. Kosten-Nutzen-Analyse

### 3.1 Kurzfristig (0-12 Monate)

| Kategorie | Kosten | Nutzen |
|-----------|--------|--------|
| **Lizenzierung** | Teil des Talent Management Bundles (ca. CHF 8-14/User/Monat) | Keine Zusatzlizenz nötig |
| **Implementation** | CHF 15'000-40'000 | - |
| **Schulung** | CHF 5'000-10'000 | - |
| **Migration** | 2-4 Wochen Aufwand | - |
| **Quick Wins** | - | Einheitliche Stellenbeschreibungen |
| | - | Gender Bias Detection (D&I) |
| | - | Professionellere Ausschreibungen |

**Kurzfrist-ROI:** Investitionsphase, noch kein messbarer Return

### 3.2 Mittelfristig (1-3 Jahre)

| Kategorie | Kosten | Nutzen |
|-----------|--------|--------|
| **Wartung** | Ca. 10-15% der Impl.-Kosten/Jahr | - |
| **Updates** | Minimal (automatisch) | - |
| **Prozesseffizienz** | - | 30-50% Zeitersparnis bei Stellenausschreibungen |
| **Datenqualität** | - | Keine redundanten/widersprüchlichen Profile |
| **Compliance** | - | Job Profile Acknowledgement (Nachverfolgung) |
| **Integration** | - | Performance Reviews nutzen gleiche Kompetenzen |
| **Career Development** | - | Karrierepfade auf strukturierten Profilen |
| **Learning** | - | Schulungen mit Kompetenzen verknüpft |

**Mittelfrist-ROI:** Break-even typischerweise nach 18-24 Monaten

### 3.3 Langfristig (3-5+ Jahre)

| Kategorie | Kosten | Nutzen |
|-----------|--------|--------|
| **Skalierung** | Minimal (cloudbasiert) | - |
| **Talent Intelligence Hub** | - | KI-gestützte Skills-Empfehlungen (ab 2025/26) |
| **Workforce Planning** | - | Skill-Gap-Analysen auf Unternehmensebene |
| **Succession Planning** | - | Nachfolgeplanung auf strukturierten Profilen |
| **Employer Branding** | - | Konsistente, professionelle Aussenwirkung |
| **Datenanalyse** | - | Reporting über Kompetenzen, Lücken, Trends |

**Langfrist-ROI:** Strategischer Wert, schwer quantifizierbar aber signifikant

## 5. Alternativen

### Alternative 1: Manuell mit Excel/SharePoint
- **Kosten:** Sehr gering
- **Integration:** Keine (Medienbrüche)
- **Fehleranfälligkeit:** Hoch
- **Empfehlung:** ❌ Nicht empfohlen

### Alternative 2: Nur Employee Central (ohne JPB)
- **Kosten:** Keine Zusatzkosten
- **Integration:** Teilweise
- **Empfehlung:** ⚠️ Nur wenn Budget sehr knapp

### Alternative 3: SharePoint-basierte Lösung (z.B. Lanteria)
- **Kosten:** CHF 5-15/User/Monat
- **Integration:** Keine native SF-Integration
- **Empfehlung:** ❌ Nicht sinnvoll wenn SF bereits im Einsatz

### Alternative 4: Anderes HCM-System (Workday, Personio)
- **Kosten:** Sehr hoch (Systemwechsel)
- **Empfehlung:** ❌ Nicht relevant - SF-Entscheidung bereits gefallen
