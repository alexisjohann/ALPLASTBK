## 1. Das Zukunftsbild: Wie funktioniert Marketing bei UBS 2028?

### 1.2 Das Zukunftsbild PRO SZENARIO

Nicht jedes Szenario führt zum gleichen Zukunftsbild. Je nach Disruption verändert sich, was «gutes Marketing» bedeutet:

```
┌─────────────────────────────────────────────────────────────────────────┐
│  S1: STATUS QUO (~15%) — «Business as Usual»                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  WIE MARKETING FUNKTIONIERT:                                            │
│  Wie heute. Kampagnen werden intern entwickelt, über klassische        │
│  Kanäle (Email, Online, Print) ausgespielt, nach Impressions und       │
│  CTR bewertet. Agenturen liefern Kreation.                             │
│                                                                         │
│  WAS «GUT» BEDEUTET:                                                    │
│  Effiziente Kampagnenproduktion, konsistente Brand Voice,              │
│  solide CTR/CPC-Optimierung. Inkrementelle Verbesserung.              │
│                                                                         │
│  BEHAVIORAL SCIENCE ROLLE:                                              │
│  Nische. Punktuelle Beratung bei einzelnen Kampagnen.                  │
│  «Nice to have», nicht systemisch integriert.                          │
│                                                                         │
│  RISIKO FÜR UBS:                                                        │
│  Wettbewerber die sich schneller bewegen (Neobanken, Fintechs)        │
│  gewinnen behavioral-affine Kund:innen.                                │
│  → Langsamer Marktanteilverlust, nicht dramatisch.                     │
│                                                                         │
│  PROVENANCE: [A] — Fortschreibung des Status Quo                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  S2: NORMALE ENTWICKLUNG (~40%) — «GenAI Adoption»                     │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  WIE MARKETING FUNKTIONIERT:                                            │
│  GenAI übernimmt Content-Produktion (Texte, Bilder, Varianten).       │
│  Marketing wird schneller und günstiger in der Ausführung.             │
│  Aber: WORÜBER kommuniziert wird, ändert sich kaum.                    │
│  Die Maschine produziert schneller — aber immer noch                   │
│  Feature-Frames, demografische Segmente, CTR-Optimierung.             │
│                                                                         │
│  WAS «GUT» BEDEUTET:                                                    │
│  Geschwindigkeit: 10× mehr Kampagnenvarianten in 1/3 der Zeit.        │
│  Personalisierung: Name, Produkt-Empfehlung, Kanal-Auswahl.          │
│  Aber: Personalisierung ≠ Behavioralisierung.                          │
│  «Lieber Herr Müller, hier ist unser Fonds» ist personalisiert.       │
│  «Herr Müller, 87% in Ihrer Situation haben vorgesorgt» ist           │
│  behavioralisiert. Grosser Unterschied.                                │
│                                                                         │
│  BEHAVIORAL SCIENCE ROLLE:                                              │
│  Wachsende Relevanz. GenAI kann Behavioral Framing nicht allein —     │
│  es braucht die RICHTIGE Anweisung. Wer GenAI mit Behavioral          │
│  Prompts füttert, bekommt bessere Ergebnisse als wer nur              │
│  «schreibe eine Email» sagt.                                           │
│  → Behavioral Science wird zum «Betriebssystem» für GenAI.            │
│                                                                         │
│  RISIKO FÜR UBS:                                                        │
│  Schneller produzieren, aber nicht besser kommunizieren.               │
│  «Industrialisierung des Mittelmasses.» Mehr Output, gleiche          │
│  Conversion-Rate. Die Effizienzgewinne werden real, die               │
│  Effektivitätsgewinne bleiben aus.                                     │
│                                                                         │
│  PROVENANCE: [A/W] — Branchentrend, JP Morgan/Persado als Referenz   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  S3: LEICHTE DISRUPTION (~30%) — «Behavioral Marketing»               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  WIE MARKETING FUNKTIONIERT:                                            │
│  Marketing versteht sich nicht mehr als «Kommunikation»               │
│  sondern als «Entscheidungsbegleitung». Der Shift:                    │
│                                                                         │
│  Alt: «Wie bringen wir Kund:innen dazu, unser Produkt zu kaufen?»    │
│  Neu: «Wie helfen wir Kund:innen, die richtige Entscheidung           │
│        für ihre Situation zu treffen?»                                  │
│                                                                         │
│  Das ist nicht nur Rhetorik. Es verändert:                             │
│  ├── WAS gemessen wird (Entscheidungsqualität, nicht Klicks)          │
│  ├── WIE segmentiert wird (Entscheidungstypen, nicht Demografie)      │
│  ├── WAS kommuniziert wird (Frames, nicht Features)                   │
│  └── WANN kommuniziert wird (Decision Windows, nicht Mediaplan)       │
│                                                                         │
│  WAS «GUT» BEDEUTET:                                                    │
│  Kund:innen treffen BESSERE Entscheidungen für sich selbst —          │
│  und wählen dabei häufiger UBS. Das ist kein Widerspruch, sondern     │
│  die Kernlogik von Behavioral Marketing: Wenn du Menschen hilfst,     │
│  gut zu entscheiden, entscheiden sie sich für den, der hilft.         │
│                                                                         │
│  BEHAVIORAL SCIENCE ROLLE:                                              │
│  Zentral. Behavioral Science ist nicht Beratung, sondern               │
│  Betriebssystem. In jedem Kampagnen-Briefing steht:                   │
│  «Welcher Entscheidungstyp? Welcher Frame? Welches Timing?»           │
│                                                                         │
│  RISIKO FÜR UBS:                                                        │
│  Organisatorischer Wandel ist schwer. Marketing-Teams müssen          │
│  umdenken. «Wir machen nicht mehr Werbung, wir designen               │
│  Entscheidungsarchitekturen.» Das erfordert neue Skills,              │
│  neue KPIs, neue Prozesse.                                             │
│                                                                         │
│  PROVENANCE: [A/H] — Richtung plausibel, Umsetzung unsicher          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│  S4: VOLLE DISRUPTION (~15%) — «System-Transformation»                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  WIE MARKETING FUNKTIONIERT:                                            │
│  Marketing verschmilzt mit Produkt, Service und Beratung.             │
│  Es gibt keine «Marketing-Abteilung» mehr im klassischen Sinn,       │
│  sondern ein «Customer Decision System» das alle Touchpoints          │
│  orchestriert — vom ersten Kontakt bis zum langfristigen               │
│  Wealth Management.                                                    │
│                                                                         │
│  Die Kund:innen-Reise ist nicht mehr linear (Awareness →              │
│  Consideration → Conversion) sondern kontextabhängig:                 │
│  Dieselbe Person bekommt verschiedene Frames, Angebote,               │
│  Touchpoints — je nach Lebenssituation, Entscheidungstyp,            │
│  aktuellem Kontext (Ψ).                                                │
│                                                                         │
│  WAS «GUT» BEDEUTET:                                                    │
│  Das System lernt kontinuierlich. Jede Interaktion verbessert         │
│  das Modell. Behavioral Insights fliessen in Echtzeit zurück.         │
│  Marketing wird zu einem selbstlernenden System das                   │
│  Entscheidungsarchitekturen personalisiert.                            │
│                                                                         │
│  BEHAVIORAL SCIENCE ROLLE:                                              │
│  Fundament. Nicht mehr beratend, sondern eingebaut.                    │
│  Behavioral Models laufen als Algorithmen. Parameter werden            │
│  kontinuierlich kalibriert. FAP wäre nicht Berater, sondern           │
│  wissenschaftlicher Partner auf System-Ebene.                          │
│                                                                         │
│  RISIKO FÜR UBS:                                                        │
│  Sehr hohe Investition. Regulatorische Unsicherheit (FINMA,           │
│  DSG). Kultureller Widerstand. Aber: wer es schafft, hat einen       │
│  fast unangreifbaren Wettbewerbsvorteil.                              │
│                                                                         │
│  PROVENANCE: [H] — Visionär, nur in Fragmenten realisiert            │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.2 Handlungsfelder in S2 (Normale Entwicklung — wahrscheinlichstes Szenario)

```
┌─────────────────────────────────────────────────────────────────────────┐
│  S2: HANDLUNGSFELDER (40% Wahrscheinlichkeit)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  GRUNDSATZ: GenAI beschleunigt die Produktion.                         │
│  UBS muss sicherstellen, dass «schneller» auch «besser» heisst.       │
│                                                                         │
│  H-S2-1: GenAI-Prompts mit Behavioral Framing anreichern              │
│  ─────────────────────────────────────────────────────                  │
│  Was:  Nicht «schreibe Email für Vorsorgeprodukt», sondern             │
│        «schreibe Email mit Loss-Frame für sicherheitsorientierte       │
│        Kund:innen 55+». GenAI Systeme mit Behavioral Briefings        │
│        füttern.                                                        │
│  Warum: GenAI ohne Behavioral Input produziert Feature-Frames.        │
│         Die Effizienz steigt, die Effektivität nicht.                  │
│  Aufwand: ◐ Mittel — Prompt-Templates + Schulung                      │
│  Provenance: [A/W] — JP Morgan macht genau das mit Persado            │
│                                                                         │
│  H-S2-2: A/B-Testing behavioral vs. konventionell                      │
│  ─────────────────────────────────────────────────                      │
│  Was:  Systematisch Behavioral Frames gegen Feature-Frames             │
│        testen. Nicht punktuell, sondern als Standard-Prozess.          │
│  Warum: Ohne Vergleich weiss niemand, ob Behavioral besser wirkt.     │
│         Der Beweis muss intern geführt werden, nicht extern.           │
│  Aufwand: ◐ Mittel — braucht Testing-Infrastruktur                    │
│  Provenance: [D/A] — A/B-Testing ist Standard, Behavioral-Variante   │
│              ist der neue Teil                                          │
│                                                                         │
│  H-S2-3: Behavioral Metrics als Ergänzung einführen                    │
│  ─────────────────────────────────────────────────                      │
│  Was:  Neben CTR/CPC auch «Behavioral Conversion Rate» messen:        │
│        Wie viele der erreichten Personen haben eine ENTSCHEIDUNG       │
│        getroffen (nicht nur geklickt)?                                  │
│  Warum: Wenn UBS nur Klicks misst, optimiert GenAI für Klicks.        │
│         Wenn UBS Entscheidungen misst, optimiert GenAI für             │
│         Entscheidungen. Die Metrik steuert das System.                 │
│  Aufwand: ◐ Mittel — Definition + Dashboard-Anpassung                 │
│  Provenance: [A] — Konzeptionell klar, Umsetzung UBS-spezifisch      │
│                                                                         │
│  H-S2-4: 1-2 Pilot-Kampagnen behavioralisieren                        │
│  ─────────────────────────────────────────────────                      │
│  Was:  Konkret 1-2 Kampagnen end-to-end behavioral redesignen:        │
│        Segment (behavioral), Frame (loss/social/default),              │
│        Timing (decision window), Journey (friction audit).             │
│  Warum: Proof of Concept. Nicht theoretisch argumentieren,             │
│         sondern zeigen: «DIESE Kampagne hat +X% Conversion,           │
│         weil wir DAS geändert haben.»                                  │
│  Aufwand: ◐ Mittel — 1-2 Kampagnen × 4-8 Wochen                      │
│  Provenance: [D/A] — Methodik [D], UBS-Anwendung [A]                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 2.3 Handlungsfelder in S3 (Leichte Disruption — der Paradigmenwechsel)

```
┌─────────────────────────────────────────────────────────────────────────┐
│  S3: HANDLUNGSFELDER (30% Wahrscheinlichkeit)                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  GRUNDSATZ: Marketing versteht sich als Entscheidungsbegleitung.       │
│  UBS muss Denken, Prozesse und Metriken grundlegend anpassen.          │
│                                                                         │
│  ALLES AUS S2 plus:                                                     │
│                                                                         │
│  H-S3-1: Behavioral Segmentation als Standard einführen                │
│  ──────────────────────────────────────────────────────                  │
│  Was:  Kund:innen nicht mehr primär nach Demografie segmentieren,      │
│        sondern nach Entscheidungstypen: Loss-Averse Delegators,        │
│        Gain-Seeking Autonomists, Status-Quo Satisficers, etc.          │
│  Warum: Die Botschaft muss zum Entscheidungstyp passen, nicht zum     │
│         Alter. Ein 60-jähriger Gain-Seeker reagiert auf ganz          │
│         andere Frames als ein 60-jähriger Loss-Averter.               │
│  Aufwand: ● Hoch — Datenmodell ändern, CRM anpassen                   │
│  Provenance: [D/A] — Methodik [D], UBS-Datenintegration [A]           │
│                                                                         │
│  H-S3-2: Framing-Playbook für alle Kampagnen-Briefings                │
│  ──────────────────────────────────────────────────────                  │
│  Was:  Ein standardisiertes Playbook: Pro Entscheidungstyp ×           │
│        Produktkategorie der optimale Frame (Loss/Gain/Social/          │
│        Default/Scarcity). Wird Teil jedes Kampagnen-Briefings.        │
│  Warum: Nicht jede Kampagne braucht einen Behavioral-Berater.         │
│         Das Playbook macht Behavioral Framing skalierbar.             │
│  Aufwand: ● Hoch — Erstellen + Schulung + Prozessintegration          │
│  Provenance: [D/A] — Framing [D], UBS-spezifisches Playbook [A]      │
│                                                                         │
│  H-S3-3: Friction Audit aller Kund:innen-Journeys                     │
│  ──────────────────────────────────────────────────────                  │
│  Was:  Jede digitale Kund:innen-Journey (Onboarding, Hauptbank-       │
│        Wechsel, Produkt-Abschluss, Key Club) systematisch auf         │
│        Behavioral Friktionen prüfen: Choice Overload, fehlende        │
│        Defaults, Uncertainty Amplification, Delayed Feedback.          │
│  Warum: L4 (Friktions-Leak) ist einer der grössten — und einer       │
│         der am einfachsten zu schliessenden.                           │
│  Aufwand: ● Hoch — alle Journeys durchgehen + Redesign                │
│  Provenance: [D/A] — Friction-Methodik [D], UBS-Journeys [A]          │
│                                                                         │
│  H-S3-4: Behavioral Training für Marketing-Team                        │
│  ──────────────────────────────────────────────────────                  │
│  Was:  Das Marketing-Team lernt, Behavioral Leaks selbst zu            │
│        erkennen. Nicht Theorie, sondern Anwendung:                     │
│        «Zeig mir deine Kampagne, ich zeige dir den Leak.»             │
│  Warum: Ohne Team-Befähigung bleibt jeder Behavioral Input             │
│         an externen Beratern hängen. Nicht skalierbar.                │
│  Aufwand: ◐ Mittel — Workshop-Serie (4-6 Halbtage)                    │
│  Provenance: [A] — Standard-Befähigungsansatz                          │
│                                                                         │
│  H-S3-5: Behavioral KPIs als Führungsinstrument                        │
│  ──────────────────────────────────────────────────────                  │
│  Was:  Marketing-Leads werden nicht nur an CTR/CPC gemessen,           │
│        sondern an Behavioral Conversion Rates und Decision             │
│        Quality Scores. Das verändert, wofür optimiert wird.            │
│  Warum: «What gets measured gets managed.» Solange CTR die             │
│         Hauptmetrik ist, wird für Klicks optimiert, nicht für          │
│         Entscheidungen.                                                │
│  Aufwand: ◐ Mittel — KPI-Definition + Dashboard + Governance          │
│  Provenance: [A/H] — Konzept klar, UBS-Umsetzung [H]                  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 3. FAP-Rolle: Wo auf welcher Ebene unterstützen?

### 3.1 Die 3 Ebenen erklärt

```
┌─────────────────────────────────────────────────────────────────────────┐
│  3 UNTERSTÜTZUNGS-EBENEN                                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  EBENE 1: PRODUKTEBENE                                                  │
│  ─────────────────────                                                  │
│  FAP arbeitet an KONKRETEN OUTPUTS:                                     │
```
