## 🔴 CRITICAL FINDINGS

### 1. M3 CROWDING-OUT WARNING ⚠️

**❌ WRONG:** "We're giving you $50 to stay"  
→ Signals distrust  
→ γ ≈ -0.20 (BACKFIRES)

**✅ RIGHT:** "We've improved your account with fair pricing"  
→ Signals respect  
→ γ ≈ +0.35 (REINFORCES)

**Source:** Fehr & Klein (2009), Güentürküen (2025)

---

### 2. LOSS AVERSION ASYMMETRY

A **$100 trust loss** feels like **$225 loss** (λ = 2.25)  
But a **$225 trust gain** feels like normal progress

**Recovery must be:**  
- 2.25x larger than initial loss, OR
- Framed as **preventing further loss** (not gaining)

**Source:** Kahneman & Tversky (1991), Benartzi (1995)

---

### 3. DEFAULTS ARE INVISIBLE WALLS

**Organ donation analogy:**
- Opt-in: 15% enroll
- Opt-out: 85% enroll
- **Difference: 5-7x effect from framing alone**

**In banking:**
- Current account = default
- Switching requires overcoming δ ≈ 0.80 inertia multiplier

**Source:** Johnson & Goldstein (2003), Madrian & Shea (2001)

---

### 4. FAIRNESS IS MULTIDIMENSIONAL

Customers evaluate fairness on **4 dimensions:**

1. **Equality** - "Am I treated same as others?"
2. **Equity** - "Is it proportional to my contribution?"
3. **Need** - "Does the bank consider my situation?"
4. **Procedure** - "Was I heard in the decision?"

Fair contract messaging must address **all 4**.

**Source:** Cappelen (2007), Fehr & Schmidt (2006)

## 📋 QUICK DECISION TREE

### For M1 (Competitive Response)

**Question:** How to retain customers against competitor X?

**Evidence Base:** Status quo bias + Switching costs + Loyalty patterns

**Steps:**
1. Estimate δ_switch_cost for your customer segment
2. Calculate competitor advantage in default switching
3. Design counter-default (offer must overcome δ ≥ 0.80 inertia)
4. Use fairness framing (σ ≈ 1.30-1.50) to lower perceived switching cost

**Key Paper:** Madrian & Shea (2001), Akerlof (1983)

---

### For M2 (Budget Allocation)

**Question:** How to get customers to allocate more budget to our products?

**Evidence Base:** Fairness frames + Mental accounting + Choice architecture

**Steps:**
1. Identify customer's mental account (liquid vs. goal vs. risky)
2. Design allocation frame respecting all 4 fairness dimensions
3. Use defaults to guide allocation (effect size: 0.50-0.75 SD)
4. Test framing: loss-frame vs. gain-frame

**Key Paper:** Cappelen (2007), Bernheim (2015), Mertens (2022)

---

### For M3 (Trust Recovery)

**Question:** How to rebuild trust after service failure?

**Evidence Base:** Reciprocity + Fair contracts + Procedural fairness

**Steps:**
1. **DO NOT:** Offer direct incentive (γ ≈ -0.15 backfire risk)
2. **DO:** Signal fair contract change (γ ≈ +0.35 reinforcement)
3. Emphasize procedural fairness (τ ≈ 0.40 effectiveness)
4. Use loss-frame: "Prevent further losses" not "Gain new benefit"
5. Demonstrate reciprocal commitment (long-term contract improvements)

**Key Papers:** Fehr & Klein (2009), Falk (2021), Kosfeld (2005)

## 🔗 COMPLEMENTARITY MATRIX

How the three models interact:

```
M1 + M2: γ ≈ +0.20
  (Competitive + Fair Budget = Stronger loyalty effect)

M1 + M3: γ ≈ -0.10
  (Lock-in + Trust Recovery = Slightly negative if forced)

M2 + M3: γ ≈ +0.35
  (Fair Budget + Trust Recovery = Strong synergy)

M1 + M2 + M3: γ ≈ +0.45
  (All three combined = Strongest synergy)
  BUT: Must sequence as M2 → M3 → M1
  (Fair Budget FIRST, then Trust, then Lock-in strengthens naturally)
```

## ⚠️ ANTI-PATTERNS TO AVOID

| Anti-Pattern | Why Bad | Effect | Fix |
|--------------|---------|--------|-----|
| M1 first + aggressive lock-in | Signals distrust | γ ≈ -0.30 | Delay lock-in until M3 complete |
| M2 + unfair allocation | Violates fairness | γ ≈ -0.25 | Test all 4 fairness dimensions |
| M3 + pure money incentive | Crowds out trust | γ ≈ -0.15 | Add fair contract signaling |
| Loss frame without context | Triggers threat response | λ ≈ 3.0 (not 2.25) | Combine with procedural fairness |
| Defaults without fairness | Feels manipulative | δ ≈ 0.40 (not 0.80) | Explain reasoning behind default |

## 🎓 HOW TO USE THIS IN YOUR PROJECT

### Step 1: Select Your Model Focus

- **M1 focused?** → Read Madrian & Shea (2001) + Johnson & Goldstein (2003)
- **M2 focused?** → Read Cappelen (2007) + Mertens (2022)
- **M3 focused?** → Read Fehr & Schmidt (2009) + Falk (2021)
- **All three?** → Start with Kahneman & Tversky (1991) as foundation

### Step 2: Extract Parameters

Use the parameter table above. For your context:
- Financial services? Use upper range (e.g., λ = 2.75 not 2.25)
- High-touch interactions? Use higher σ values (1.5 not 1.3)
- Trust recovery critical? Use τ = 0.45 not 0.35

### Step 3: Test Locally

Run small pilot with 100-1000 customers using recommended parameters.

### Step 4: Scale

DellaVigna & Linos (2022) show effects persist at scale, but median effect 0.30 SD (not theoretical 0.43 SD).

### Step 5: Measure Crowding-Out

Monitor for γ < 0 signals:
- Customers interpreting fairness improvements as manipulation?
- Loyalty declining despite fair offerings?
- Switching rate up despite defaults?

→ Adjust fairness framing if γ appears negative
