#### Kapitel 12: Willingness-Funktion (CORE-READY) — ●●●

**EBF-Konzept:** WAX-Formel mit Schwelle θ

**Die Master-Gleichung:**
```
WAX = φ × min{U*} + (1-φ) × Σ U*

Handlung ↔ WAX ≥ θ
```

**Der φ-Parameter (Denkstil):**
- φ → 1: "Verbundenes Denken" (schwächste Dimension dominiert)
- φ → 0: "Getrenntes Denken" (additive Bewertung)

**Relevanz für BEATRIX:**
- Erklärt die **Intentions-Verhaltens-Lücke** ("Ich will, aber...")
- Quantifiziert Barrieren und Erleichterungen
- **Neuheit:** Formale Modellierung des "Knowing-Doing Gap"

**Zitat aus Kapitel 12:**
> "Thomas weiss, dass Sport gesund ist. Aber WAX < θ → er handelt nicht."

### c) Machbarkeit und Risikoreduktion

**Kernfrage:** Ist das Projekt in 125 Stunden realisierbar? Wie werden Risiken adressiert?

#### Kapitel 10: LLM Monte Carlo Validierung — ●●●

**EBF-Konzept:** Parameter-Schätzung via LLM Monte Carlo (LLMMC)

**Die Methodik (Appendix UNMAPPED_LLM):**
1. N=100+ LLM-Durchläufe pro Parameter
2. Verteilungsanalyse (mean, std, 95% CI)
3. Epistemic Tags: [EMP], [THR], [EXP]
4. Human-in-the-Loop Kalibrierung

**Relevanz für BEATRIX:**
- Ermöglicht **automatisierte Parameterschätzung** ohne aufwändige Primärforschung
- 95% Konfidenzintervalle für alle Vorhersagen
- **Machbarkeit:** LLM-MC ist in 125h durchführbar; klassische RCTs wären unmöglich

**Technische Details aus Kapitel 10:**
> "Die Γ-Matrix (144×144 Interaktionen) wurde via LLMMC mit N=500 geschätzt. Mean estimation time: 0.3s per parameter."

#### Kapitel 17: R-Score und Entscheidungsregeln — ●●●

**Das R-Score System:**
```
R(Archetyp, Kontext) = Σ αᵢ × Fit_i(Archetyp, Kontext)

R > 0.7 + EMP → GO
R > 0.7 + THR/EXP → PILOT
0.4 < R ≤ 0.7 → PILOT
R ≤ 0.4 → NO-GO
```

**Relevanz für Risikoreduktion:**
- Klare Entscheidungsregeln für Interventionen
- Keine "Bauchentscheidungen" mehr
- **Machbarkeit:** R-Score Berechnung ist automatisierbar
