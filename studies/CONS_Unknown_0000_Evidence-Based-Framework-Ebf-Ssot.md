## Definition

Das Evidence-Based Framework (EBF) ist ein integratives wissenschaftliches Framework für ökonomisches und soziales Verhalten, entwickelt von FehrAdvice & Partners AG in Zusammenarbeit mit Prof. Ernst Fehr (Universität Zürich).

## EBF Grundaxiome

1. **Empirische Fundierung** — Parameter basieren auf Papers
2. **BBB 4-Tier Hierarchie** — Literature → LLMMC → Empirical → Expert
3. **Referentielle Integrität** — Jeder Parameter hat Registry-Eintrag
4. **Begründete Komplementarität** — γ ≠ 0 nur mit Paper-Quelle
5. **Default Additivität** — Komplementarität nur wenn nötig
