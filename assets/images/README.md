# Image Library

> Bildmaterial und Foto-Richtlinien für FehrAdvice

---

## 📸 Foto-Richtlinien

### ✅ Do's

| Kriterium | Beschreibung |
|-----------|--------------|
| **Authentisch** | Echte Situationen, nicht gestellt |
| **Professionell** | Hochwertige Aufnahmen |
| **Divers** | Vielfältige Menschen und Situationen |
| **Natürlich** | Leicht entsättigt, keine Filter |
| **Kontextreich** | Zeigt Arbeits-/Entscheidungssituationen |

### ❌ Don'ts

- Keine gestellten "Business-Handshake"-Fotos
- Keine übertrieben glücklichen Stock-Fotos
- Keine stark bearbeiteten/gefilterten Bilder
- Keine Bilder mit Wasserzeichen
- Keine Bilder unter 300 dpi für Print

---

## 🎨 Bildbearbeitung

| Parameter | Wert |
|-----------|------|
| Sättigung | Leicht reduziert (-10%) |
| Kontrast | Normal bis leicht erhöht |
| Helligkeit | Natürlich |
| Farbtemperatur | Neutral bis leicht kühl |
| Filter | Keine |
| Rahmen | Kein Rahmen |
| Ecken | Keine Abrundung |

## 📁 Ordnerstruktur

```
images/
├── photos/              # Genehmigte Fotos
│   ├── team/            # Team-Fotos
│   ├── office/          # Büro/Arbeitsumgebung
│   ├── workshops/       # Workshop-Situationen
│   ├── presentations/   # Präsentations-Situationen
│   └── clients/         # (Anonymisierte) Kunden-Situationen
│
├── illustrations/       # Illustrationen
│   ├── concepts/        # Konzept-Visualisierungen
│   └── diagrams/        # Erklärende Diagramme
│
└── examples/            # Beispiele für Richtlinien
    ├── approved/        # ✅ Genehmigte Beispiele
    └── rejected/        # ❌ Abgelehnte Beispiele (als Referenz)
```

---

## 📐 Bildformate

| Verwendung | Format | Mindestauflösung |
|------------|--------|------------------|
| Print | TIFF, PNG | 300 dpi |
| Web | JPEG, PNG, WebP | 72 dpi, min. 1200px breit |
| Social Media | JPEG, PNG | Je nach Plattform |
| Präsentationen | PNG, JPEG | 150 dpi |
