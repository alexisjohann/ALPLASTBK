# Infographic Templates

> Templates für Datenvisualisierung und Infografiken

---

## 🎨 Design-Prinzipien

| Prinzip | Beschreibung |
|---------|--------------|
| **Klarheit** | Eine Kernaussage pro Infografik |
| **Hierarchie** | Wichtigstes zuerst, visuell hervorgehoben |
| **Konsistenz** | FehrAdvice-Farben und -Schriften |
| **Einfachheit** | Reduziert auf das Wesentliche |
| **Barrierefreiheit** | Kontrastreiche Farben, Alternativtexte |

---

## 📁 Template-Typen

### 1. Prozess-Diagramme (`process/`)

```
Für: Schritt-für-Schritt-Abläufe, Workflows, Customer Journeys

┌─────┐    ┌─────┐    ┌─────┐    ┌─────┐
│  1  │ → │  2  │ → │  3  │ → │  4  │
└─────┘    └─────┘    └─────┘    └─────┘
```

**Dateien:**
- `process-horizontal.svg` - Horizontaler Ablauf
- `process-vertical.svg` - Vertikaler Ablauf
- `process-circular.svg` - Kreisförmiger Ablauf

---

### 2. Vergleichs-Matrizen (`comparison/`)

```
Für: Vor/Nach, Option A vs B, Feature-Vergleiche

┌──────────┬─────────┬─────────┐
│          │ Option A│ Option B│
├──────────┼─────────┼─────────┤
│ Kriterium│   ✓     │   ✗     │
└──────────┴─────────┴─────────┘
```

**Dateien:**
- `comparison-table.svg` - Tabellen-Format
- `comparison-side-by-side.svg` - Nebeneinander
- `comparison-venn.svg` - Venn-Diagramm

---

### 3. Zeitleisten (`timeline/`)

```
Für: Historische Entwicklungen, Projektpläne, Meilensteine

    ●────────●────────●────────●
   2020     2022     2024     2026
```

**Dateien:**
- `timeline-horizontal.svg`
- `timeline-vertical.svg`
- `timeline-milestone.svg`

---

### 4. Hierarchie/Organigramme (`hierarchy/`)

```
Für: Organisationsstrukturen, Entscheidungsbäume, Taxonomien

           ┌─────┐
           │ CEO │
           └──┬──┘
       ┌──────┼──────┐
    ┌──┴──┐ ┌─┴─┐ ┌──┴──┐
    │ CFO │ │COO│ │ CTO │
    └─────┘ └───┘ └─────┘
```

**Dateien:**
- `hierarchy-org-chart.svg`
- `hierarchy-tree.svg`
- `hierarchy-pyramid.svg`
