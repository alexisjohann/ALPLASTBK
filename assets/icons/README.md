# Icon Library

> Konsistente Icon-Bibliothek für FehrAdvice

---

## 🎨 Icon-Spezifikationen

| Eigenschaft | Wert |
|-------------|------|
| Stil | Liniensymbole (Outline) |
| Strichstärke | 1.5–2 pt |
| Farbe | Dunkelblau (`#024079`) oder Dunkelgrau (`#25212A`) |
| Standardgrösse | 24 × 24 px |
| Format | SVG (primär), PNG (Fallback) |

## 📏 Grössenvarianten

Für jedes Icon werden folgende Grössen benötigt:

```
icon-name/
├── icon-name-16.svg     # Kleine Anwendungen
├── icon-name-24.svg     # Standard
├── icon-name-32.svg     # Mittlere Anwendungen
├── icon-name-48.svg     # Grosse Anwendungen
└── icon-name-64.svg     # Hero-Bereiche
```
