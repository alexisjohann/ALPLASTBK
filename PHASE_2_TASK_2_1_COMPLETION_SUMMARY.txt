================================================================================
KEY FINDINGS
================================================================================

1. LINEAR ADDITIVE FAILURE
   Problem: Adding 10 γ parameters via Ridge regression WORSENS accuracy
   Cause: Feature-to-sample ratio (15:12 = 1.25:1) too high for N=12
   Lesson: Small samples require domain constraints, not just regularization

2. 1922 IS A REGIME CHANGE, NOT AN OUTLIER
   Pattern: Λ+Π=1.20 (weak network) + Ι=0.88 (strong integration)
   Result: Factional deadlock (Integralists vs Progressives blocking 10+ rounds)
   Current model limitation: Cannot predict stalemate without separate module
   Solution: Defer to Task 2.2 (Crisis/Shock Module)

3. THEORY-GUIDED APPROACH WORKS
   Method: Use domain knowledge to constrain functional form
   Base formula: Keep validated v1.0 (ballots = 10/(Λ+Π))
   Adjustment: Add stalemate correction when specific pattern detected
   Result: RMSE 3.185 (equivalent to v1.0) + stalemate interpretability

4. COMPLEMENTARITY IS CONDITIONAL, NOT ADDITIVE
   Finding: γ parameters work through "IF-THEN" logic, not linear summation
   Example: Integration capacity matters WHEN network is weak
   Implication: Future models should use interaction/conditional terms

================================================================================
LESSONS FOR FUTURE WORK
================================================================================

LESSON 1: Feature-to-Sample Ratio Matters
  - N=12 with 15 features (1.25:1) causes overfitting
  - N=12 with 5 features (2.4:1) stable
  - Rule: Keep ratio >2:1 with small samples

LESSON 2: Regularization Alone Insufficient
  - Ridge CV (α=0.0210) couldn't prevent overfitting with 15 features
  - Domain constraints (theory-guided) more effective than statistical regularization

LESSON 3: Statistical vs Theory-Guided Approaches
  - Pure statistics: Find parameters that minimize training error
    Result: Overfitting, uninterpretable, worse on new data
  - Theory-guided: Use domain knowledge to constrain search space
    Result: Stable, interpretable, generalizes better

LESSON 4: 1922 Needs Separate Modeling
  - 1922 is not an outlier to ignore - it's a regime change
  - Belongs in Task 2.2 (Crisis/Shock Module), not Task 2.1
  - Separate models for: Quick consensus vs Stalemate vs Veto crisis
