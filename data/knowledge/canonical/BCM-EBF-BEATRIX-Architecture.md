# Wie BCM, EBF und BEATRIX zusammenhängen — SSOT

**Status:** Canonical Definition | **Tags:** architecture, ebf, bcm, beatrix, ssot, canonical

## ACHTUNG: Drei verschiedene Dinge

| Konzept | Typ | Beschreibung |
|---------|-----|--------------|
| **EBF** | Framework (Theorie) | Das übergeordnete wissenschaftliche Framework |
| **BCM** | Modell (Teil von EBF) | Behavioral Change Model — erklärt Verhaltenswandel |
| **BEATRIX** | Produkt (Software) | Web-Plattform die EBF + BCM implementiert |

**Diese drei sind NICHT austauschbar!**

## Die Hierarchie

```
EBF (Evidence-Based Framework)
 │  Zentrale Gleichung: θ = f(Ψ, 10C)
 │
 ├── BCM (Behavioral Change Model)
 │    P = Willingness × Ability × Context
 │
 ├── 10C CORE Framework
 │    WHO, WHAT, HOW, WHEN, WHERE, AWARE, READY, STAGE, HIERARCHY, EIT
 │
 ├── 8 Ψ-Dimensionen (Kontext)
 │    Ψ_I, Ψ_S, Ψ_C, Ψ_K, Ψ_E, Ψ_T, Ψ_M, Ψ_F
 │
 ├── FEPSDE (6 Utility-Dimensionen)
 │    Financial, Emotional, Physical, Social, Digital, Ecological
 │
 ├── Parameter Registry (64+ Parameter)
 ├── Theory Catalog (153 Theorien)
 ├── Case Registry (852 Cases)
 └── Paper Database (2'347 Papers)
```

## Was sie NICHT sind

### EBF ist NICHT:
- Ein Chatbot (das ist BEATRIX)
- Ein einzelnes Modell (BCM ist ein Modell, EBF ist das Framework)
- Software (EBF ist die wissenschaftliche Grundlage)

### BCM ist NICHT:
- Das gesamte Framework (das ist EBF)
- "Behavioral Competence Model" ❌
- "Behavioral Context Model" ❌
- BCM = **Behavioral Change Model** ✅

### BEATRIX ist NICHT:
- Die Theorie (das ist EBF)
- Das Modell (das ist BCM)
- BEATRIX = Web-Plattform die beides implementiert
- Steht für: Behavioral Economics AI Technology for Research and Implementation eXcellence
- IMMER in Grossbuchstaben

## Wie sie zusammenspielen

```
Nutzer → BEATRIX (Chat)
           ↓
     EBF-Wissen abrufen
           ↓
     10C + Ψ + BCM + Parameter + Cases
           ↓
     Strukturierte Antwort
```
