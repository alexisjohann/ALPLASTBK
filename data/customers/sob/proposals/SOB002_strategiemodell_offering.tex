\section{Gegenstand der Offerte}
% =============================================================================

Entwicklung eines integrierten SOB-Strategiemodells in BEATRIX (EBF), das alle bisher erarbeiteten Elemente konsolidiert (Vision, Mission) und die Implementierung der Gesamtstrategie auf verschiedenen Dimensionen projektieren kann.

% =============================================================================
\section{Leistungsumfang}
% =============================================================================

\subsection{Inbegriffen}

\begin{enumerate}
    \item Integration Vision + Mission in Strategiearchitektur
    \item Stakeholder-Mapping (VR, GL, Bereichsleiter, MA)
    \item Gap-Analyse bestehender Strategiedokumente
    \item Entwicklung SOB-Strategiemodell in BEATRIX/EBF
    \item Definition der Strategie-Dimensionen
    \item Kalibrierung der Verhaltensparameter
    \item Entwicklung Projektions-Tool / Dashboard
    \item Bereichsspezifische Übersetzung (4--5 Factsheets)
    \item GL-Workshop zur Validierung
    \item Finale Dokumentation (Strategie-Handbuch)
\end{enumerate}

\subsection{Nicht inbegriffen}

\begin{itemize}
    \item Detaillierte Massnahmenplanung pro Bereich
    \item Change Management / Kommunikationskonzept
    \item IT-Implementierung ausserhalb BEATRIX
    \item Schulungen / Trainings
    \item Quantitative Marktforschung
\end{itemize}
