## 1. Executive Summary

Diese Gap-Analyse konsolidiert systematisch alle identifizierten Lücken zwischen dem IST-Zustand der SOB und dem SOLL-Zustand gemäss Eignerstrategie (DOC-003), eigener Strategieambition (DOC-002/DOC-009) und verhaltensökonomischem Optimum (EBF-Benchmark). Die Analyse identifiziert **47 diskrete Gaps** in 7 Dimensionen, priorisiert nach Dringlichkeit und Hebelwirkung.

**Kernbefund:** Die SOB hat drei fundamentale Gap-Cluster:

```
┌─────────────────────────────────────────────────────────────────────────┐
│  3 GAP-CLUSTER DER SOB                                                  │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  CLUSTER A: STRATEGIE-ARCHITEKTUR                                       │
│  «Die SOB weiss, WAS sie sein will, aber nicht WIE»                    │
│  → 15 Gaps (Pyramide, Ziele, Massnahmen, KPIs)                         │
│  → Dringlichkeit: HOCH (Konzessionserneuerung 2029)                    │
│                                                                         │
│  CLUSTER B: ORGANISATIONS-INTEGRATION                                   │
│  «Jeder GB ist für sich stark, aber die GBs arbeiten nicht zusammen»   │
│  → 18 Gaps (Silos, Prozesse, Steuerung, Kultur)                        │
│  → Dringlichkeit: SEHR HOCH (γ ≈ 0.30 vs. Ziel 0.50)                  │
│                                                                         │
│  CLUSTER C: STEUERUNGS-BLINDHEIT                                        │
│  «Was nicht gemessen wird, wird nicht gemanagt»                        │
│  → 14 Gaps (KPIs, Leading Indicators, Feedback-Loops)                  │
│  → Dringlichkeit: HOCH (8/10 Probleme ohne KPI)                        │
│                                                                         │
│  WECHSELWIRKUNG:                                                        │
│  A × B × C verstärken sich gegenseitig:                                │
│  Keine Ziele (A) → keine Zusammenarbeitsziele (B) → nicht messbar (C)  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 2. Methodik

### 2.1 Gap-Definition

Ein Gap wird definiert als messbare Differenz zwischen:

| Vergleich | IST-Quelle | SOLL-Quelle | Bezeichnung |
|-----------|-----------|-------------|-------------|
| IST vs. Eignerstrategie | DOC-001 bis DOC-011 | DOC-003 | **Governance-Gap** |
| IST vs. Eigene Strategie | DOC-001 bis DOC-011 | DOC-002, DOC-009 | **Ambitions-Gap** |
| IST vs. EBF-Benchmark | 10C-Diagnose | EBF Best Practice | **Behavioral Gap** |

### 2.2 Priorisierungs-Matrix

Jeder Gap wird bewertet nach:

| Kriterium | Skala | Gewicht |
|-----------|-------|---------|
| **Dringlichkeit** | 1-5 (5 = sofort) | 30% |
| **Hebelwirkung** | 1-5 (5 = grösster Impact) | 35% |
| **Machbarkeit** | 1-5 (5 = sofort umsetzbar) | 20% |
| **Evidenzstärke** | 1-5 (5 = mehrere Quellen) | 15% |

**Prioritäts-Score = 0.30×D + 0.35×H + 0.20×M + 0.15×E**

### 2.3 Quellen-Triangulation

```
┌─────────────────────────────────────────────────────────────────────────┐
│  EVIDENZ-QUALITÄT PRO GAP                                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ★★★★★  Quantitativ + Qualitativ + Strategisch (≥3 Quellen)            │
│  ★★★★   Quantitativ + Qualitativ ODER 2+ Quellen                       │
│  ★★★    Eine starke Quelle (z.B. MA-Umfrage N=771)                    │
│  ★★     Qualitative Einzelquelle                                       │
│  ★      Abgeleitet / EBF-Inference                                     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
