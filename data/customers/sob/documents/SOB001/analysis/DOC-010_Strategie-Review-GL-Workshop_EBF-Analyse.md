## 1. Executive Summary

Dieses Dokument ist der **Workshop-Leitfaden** für die GL-Klausur vom 12.09.2025. Es definiert die Zielsetzung des Strategie-Reviews und strukturiert den Workshop in 3 Phasen: Ausgangslage → Zielrichtung → Konkretisierung. Es ist kein Ergebnis-Dokument, sondern ein **Prozess-Dokument**.

**Kernbefund:** Der Workshop bestätigt, dass die GL die gleichen Defizite sieht wie DOC-001 und DOC-009 -- aber die Workshop-Fragen offenbaren auch die **Unsicherheit der GL**: Viele Fragen enden mit «?» und «Welche Themen wollen wir im Rahmen der Strategie adressieren?». Die GL sucht Orientierung, nicht nur Bestätigung.

**Strategische Relevanz:** Mittel. Das Dokument ist prozessual wichtig (zeigt WIE die GL den Review angeht), aber inhaltlich wenig Neues gegenüber DOC-009. Der zentrale Mehrwert: Das Konzept **«Meine Lieblingsbahn»** taucht hier erstmals als potentielle Vision auf.

## 4. Verhaltensökonomische Analyse

### 4.1 «Meine Lieblingsbahn» -- Identitätsökonomische Bewertung

```
┌─────────────────────────────────────────────────────────────────────────┐
│  VISION «MEINE LIEBLINGSBAHN» — ANALYSE                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  STÄRKEN:                                                              │
│  ✅ "MEINE" → Besitztumseffekt (Endowment), persönliche Beziehung     │
│  ✅ "LIEBLINGS-" → Emotionale Bindung, nicht nur funktional            │
│  ✅ "BAHN" → Klare Identität: Wir sind eine Bahn (keine Plattform)    │
│  ✅ Einfach, merkbar, differenzierend                                  │
│  ✅ Funktioniert nach innen (MA-Stolz) UND aussen (Kunden)            │
│                                                                         │
│  RISIKEN:                                                              │
│  ⚠️  "Lieblings-" impliziert Vergleich — mit WEM?                     │
│      → Nur mit SBB sinnvoll, aber will SOB diesen Vergleich?          │
│  ⚠️  "Meine" könnte als possessiv/exklusiv wirken                     │
│  ⚠️  Kann banal klingen wenn nicht mit Substanz unterlegt              │
│                                                                         │
│  EBF-BEWERTUNG:                                                        │
│  → Social Identity Theory: "Meine Bahn" aktiviert In-Group-Identität  │
│  → Endowment Effect: "Meine" erzeugt Besitzgefühl                    │
│  → Construal Level: Konkret ("Bahn") statt abstrakt ("Mobilität")     │
│  → PASST zu SOB-DNA: Klein, persönlich, nah                           │
│                                                                         │
│  EMPFEHLUNG: Konzept hat Potential, braucht aber:                      │
│  1. Operationalisierung: Was macht eine "Lieblingsbahn" konkret?       │
│  2. Messung: Woran merken wir, dass wir es sind?                       │
│  3. Substanz: Was unterscheidet uns faktisch?                          │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
