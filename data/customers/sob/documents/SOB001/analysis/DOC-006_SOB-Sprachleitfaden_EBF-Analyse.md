## 1. Executive Summary

Der Sprachleitfaden ist ein **gut strukturiertes, professionelles Corporate-Language-Dokument** aus 2016 (letzte Anpassung 2023). Er definiert klare Regeln für Schreibweisen, Typografie, Korrespondenz und Hausregeln. Verhaltensökonomisch ist er ein **Kongruenz-Signal**: Die Art, wie die SOB schreibt, reflektiert ihr Selbstverständnis als nahbare, persönliche, schweizerische Bahngesellschaft.

**Kernbefund:** Der Leitfaden zeigt eine Organisation, die **Einheitlichkeit anstrebt aber Flexibilität betont** -- ein Spannungsfeld, das sich auch in der Strategiearbeit widerspiegelt. Für FehrAdvice relevant: Der Leitfaden definiert den **Kommunikationston**, in dem jegliche Strategie-Deliverables formuliert sein müssen, die nach aussen oder intern bei der SOB verwendet werden.

**Strategische Relevanz:** Mittel. Kein direktes Strategie-Dokument, aber **SSOT für den Kommunikationsstil** der SOB -- relevant für die Formulierung von Interventionen, Change-Kommunikation und Deliverables.

## 3. Kommunikationsstil der SOB -- Verhaltensökonomische Analyse

### 3.1 Die 5 Sprachprinzipien

Der Leitfaden definiert (Kapitel 3) fünf Kommunikationsprinzipien:

```
┌─────────────────────────────────────────────────────────────────────────┐
│  SOB-KOMMUNIKATIONSPRINZIPIEN                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. KURZ HALTEN                                                        │
│     → Keine Schachtelsätze, Verben zusammenziehen                      │
│                                                                         │
│  2. VERSTÄNDLICH SCHREIBEN                                             │
│     → Verben statt Nomen, einfache Formen                              │
│     → "beginnen" statt "in Angriff nehmen"                             │
│                                                                         │
│  3. KONKRET UND DIREKT                                                 │
│     → Keine Floskeln, sparsam mit Superlativen                        │
│     → "Wir freuen uns" statt "Wir würden uns freuen, ... zu dürfen"   │
│                                                                         │
│  4. AKTIV UND POSITIV                                                  │
│     → Aktivsätze statt Passiv                                         │
│     → Positive Formulierungen bleiben länger im Gedächtnis            │
│                                                                         │
│  5. AUF DAS GEGENÜBER EINGEHEN                                        │
│     → Intern: Du-Kultur                                                │
│     → Extern: Sie (Ausnahme: Social Media → Du)                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 Verhaltensökonomische Interpretation

| Prinzip | EBF-Konzept | Implikation |
|---------|-------------|-------------|
| Kurz halten | **Cognitive Load Reduction** | Reduziert Processing Fluency-Barrieren |
| Verständlich | **System 1 Kompatibilität** | Einfache Texte werden schneller verarbeitet |
| Konkret | **Construal Level Theory** | Konkretes wird als näher, relevanter empfunden |
| Aktiv/Positiv | **Framing Effect** | Positive Frames erhöhen Akzeptanz |
| Gegenüber | **Social Proximity** | Du-Kultur → geringere soziale Distanz → mehr Vertrauen |

**Beobachtung:** Die SOB formuliert (bewusst oder unbewusst) **verhaltensökonomisch korrekte** Kommunikationsprinzipien. Dies ist eine Stärke, auf der FehrAdvice aufbauen kann.

### 3.3 Say-Do Gap in der Kommunikation

Der Leitfaden **postuliert** einen Kommunikationsstil -- aber die MA-Umfrage (DOC-004) zeigt:

| Was der Leitfaden sagt | Was die MA-Umfrage zeigt | Gap |
|------------------------|--------------------------|-----|
| «Offen und ehrlich kommunizieren» | Transparenz & Offenheit: 73 Punkte (-1) | Moderat |
| «Persönlicher Kontakt» | Soziale Nachhaltigkeit: 84 Punkte | Klein |
| «Einheitlicher Auftritt» | Gemeinsame Ausrichtung: 70 Punkte (-3) | **Gross** |
| «Auf Bedürfnisse eingehen» | Kundenorientierung: nicht separat gemessen | Unklar |

**Kernproblem:** Der Leitfaden definiert einen einheitlichen **sprachlichen** Auftritt, aber die Organisation hat keinen einheitlichen **strategischen** Auftritt (Silo-Problematik).

1. **Infrastrukturbetreiberin** (ISB)
2. **Eisenbahnverkehrsunternehmerin** (EVU)
3. **Rollmaterialsinstandhalterin**

## 5. Verhaltensökonomische Beobachtungen

### 5.1 Identitäts-Signale im Sprachleitfaden

Der Leitfaden verrät viel über das **Selbstverständnis** der SOB:

| Signal | Textbeleg | EBF-Interpretation |
|--------|-----------|-------------------|
| **Nahbarkeit** | «persönlichen Kontakt pflegt» | Soziale Identität: «Wir sind nahbar» |
| **Eigenständigkeit** | «ein eigenständiges Unternehmen» | Abgrenzung von SBB |
| **Schweizerisch** | Billett, Perron, öV | Helvetische Identität als Differenzierungsmerkmal |
| **Offenheit** | «offen und ehrlich kommunizieren» | Vertrauens-Signal |
| **Flexibilität** | «nicht in Stein gemeisselt» | Vermeidung von Rigidität |
| **Weiblich** | «die SOB», ISB, EVU in weiblicher Form | Gender-Bewusstsein (seit 2016!) |

### 5.3 Kulturelle Einordnung

| Dimension | SOB-Positionierung | Implikation |
|-----------|-------------------|-------------|
| **Formalität** | Mittel (Du intern, Sie extern) | Typisch Schweizer öV |
| **Hierarchie** | Flach signalisiert (Du-Kultur) | Aber: VR-Beilage = formell |
| **Innovation** | Konservativ (Arial, kein ß, bewährt) | Konsistent mit Bahnbranche |
| **Schweizerisch** | Stark (Billett, Perron, Guillemets) | Differenzierung vs. SBB/DB |
| **Digitalisierung** | Niedrig (E-Mail-Fokus, kein Social-Media-Tiefe) | Update-Bedarf |

## 6. Implikationen für FehrAdvice-Projekt

### 6.1 Für interne SOB-Deliverables

```
PFLICHT:
✅ Du-Form verwenden (intern)
✅ Einfache, aktive, positive Sprache
✅ SOB-Terminologie korrekt (die SOB, öV, Billett, Perron)
✅ Guillemets «» in professionellen Dokumenten
✅ Festabstand bei Tausenderstellen (kein Apostroph!)
✅ Gendering: Doppelnennung oder neutrale Form (NICHT Doppelpunkt)
✅ Produktnamen korrekt (Voralpen-Express MIT, Treno Gottardo OHNE Bindestrich)
```

### 6.2 Für Change-Kommunikation

Der Sprachleitfaden zeigt: Die SOB legt Wert auf **einheitliches Auftreten**. Dies ist eine Chance für den Kulturwandel:

1. **Sprache als Hebel für Kulturwandel:** Neue Werte-Sprache (aus DOC-011) muss in den Leitfaden integriert werden
2. **Say-Do Gap adressieren:** Der Leitfaden sagt «offen kommunizieren» -- die MA-Umfrage zeigt Defizite bei Transparenz (73 Punkte)
3. **Silo-Sprache identifizieren:** Untersuchen, ob verschiedene Bereiche verschiedene «Dialekte» sprechen (Infrastruktur vs. Fernverkehr vs. RPV)
4. **Aktualisierungsbedarf:** Der Leitfaden ist 10 Jahre alt und braucht ein Update -- dies kann mit dem Strategieprozess verknüpft werden

### 6.3 FehrAdvice-Empfehlung

**Der Sprachleitfaden ist kein Strategiedokument, aber ein Kulturdokument.** Er zeigt, welche Identität die SOB nach aussen kommunizieren will. Für das Beratungsprojekt bedeutet das:

- **Deliverables im SOB-Ton formulieren** (nicht im FA-Ton)
- **Sprachleitfaden als Kulturindikator nutzen:** Die Diskrepanz zwischen dem Anspruch (nahbar, offen, einheitlich) und der Realität (Silos, Say-Do Gap, sinkende Ausrichtung) ist ein konkreter Ansatzpunkt für den Kulturwandel
- **Update des Leitfadens als Quick Win vorschlagen:** Zeigt dem VR, dass der Kulturwandel auch in konkreten, sichtbaren Artefakten stattfindet
