## 5. Verhaltensökonomische Kernbefunde

### 5.1 Strategische Identitätslücke (H1 -- Zentraler Befund)

```
┌─────────────────────────────────────────────────────────────────────────┐
│  KERNDIAGNOSE: Die Strategie ist NICHT handlungsleitend                 │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  IST-Zustand:                                                           │
│  Strategie = Vision + «Rollen der Geschäftsbereiche»                    │
│  → Sachlich (= vereinfachte Geschäftsaufträge)                          │
│  → Ohne konkrete Zielsetzungen und Massnahmen                           │
│  → Kaum handlungsleitend                                                │
│  → Keine motivations-/identitätsstiftenden Elemente                     │
│                                                                         │
│  EBF-Interpretation:                                                    │
│  Die Strategie adressiert WHAT (Geschäftsaufträge), aber NICHT:         │
│  → WHO: Für wen genau? (Segmente fehlen)                                │
│  → HOW: Wie interagieren die Bereiche? (Silos, H4)                     │
│  → AWARE: Ist die Strategie bewusst? (Nein -- «kaum handlungsleitend»)  │
│  → READY: Sind die MA bereit? (Ja -- aber ohne Richtung)               │
│                                                                         │
│  → Hohe Motivation (S2) × fehlende Richtung (H1) = ungenutztes         │
│    Potenzial. Die Energie ist da, aber sie hat kein Ziel.               │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**EBF-Parameter:**
- Awareness A(Strategie) ≈ 0.3 (niedrig -- Strategie ist bekannt aber nicht handlungsrelevant)
- Willingness W(Veränderung) ≈ 0.8 (hoch -- Motivation und Einsatzbereitschaft vorhanden)
- **Gap: W - A = 0.5** → Grösste Hebelwirkung liegt in AWARENESS, nicht in MOTIVATION

### 5.2 Silo-Effekt als Komplementaritäts-Problem (H4)

```
QRSU-Organisation:
  Stärke: Spezialisierung → hohe fachliche Tiefe
  Schwäche: γ(Bereich_i, Bereich_j) ≈ 0 (keine Komplementarität)

  → Wissenskonzentration auf Einzelpersonen = Single Point of Failure
  → Doppelspurigkeiten = negative Komplementarität γ < 0
  → Langwierige Entscheidungsprozesse = Koordinationskosten

  Zielbild (aus Dokument):
  "Kombination aus zentraler Steuerung, dezentraler Umsetzung
   und agiler Fachgruppenarbeit"
  → γ(zentral, dezentral) > 0 angestrebt
```

### 5.3 Formalisierte Führungsprozesse als Default-Problem (H2)

```
IST: Zielvereinbarung, Leistungsbeurteilung, Führungsgespräch
     = formalistisch ausgestaltet, entfalten wenig Wirkung

EBF-Interpretation:
  → Prozesse existieren (Ψ_I hoch), aber als "leere Defaults"
  → Compliance ist hoch (Formulare werden ausgefüllt)
  → Wirkung ist niedrig (kein Verhaltenseffekt)
  → Klassisches "Form over Substance" Problem

  κ_default(Führungsprozess) = 0.9  (Compliance hoch)
  Wirkung(Führungsprozess) ≈ 0.2   (Impact niedrig)
  → Default-Effektivitäts-Lücke = 0.7
```

### 5.4 Digitale Transformation als Awareness-Challenge (H3)

```
"Herausforderung, auch Mitarbeitende in der Linie zu erreichen,
 die Zielsetzung vermitteln und spürbar machen."

EBF-Interpretation:
  → Problem ist NICHT technisch (Projekte laufen)
  → Problem ist AWARE: Linienmitarbeitende sehen den Nutzen nicht
  → Problem ist Ψ_F: Physische Distanz (Büro vs. Zug/Werkstatt)

  A(Digi-Transform | Büro) ≈ 0.7
  A(Digi-Transform | Linie) ≈ 0.3
  → Awareness-Gap nach Arbeitsort = 0.4
```

### 5.5 CD/CI-Inkonsistenz als Identitäts-Problem (H5)

```
Zwei Designs × Zwei Web-Auftritte × Zwei Zugdesigns (silber/kupfer)
  → Inkonsistente Markenidentität
  → Kognitiver Konflikt bei Kunden und Mitarbeitenden

EBF-Interpretation:
  → U_IDN (Identitäts-Utility) wird geschwächt durch Fragmentierung
  → Kein klares "Wer sind wir?" nach aussen
  → Verbindung zu H1: Strategie fehlt → Identität fragmentiert → CD/CI inkonsistent

  Die Kausalkette: Fehlende Strategie → Fehlende Identität → Inkonsistenter Auftritt
```

## 10. Fazit

Der 100-Tage-Bericht ist ein **strategisch wertvolles Dokument** für SOB002, weil er die Perspektive des neuen CEO nach systematischer Beobachtung wiedergibt -- mit einer seltenen Mischung aus Wohlwollen (Stärken anerkennen) und Klarheit (Schwächen benennen).

**Die zentrale Erkenntnis für SOB002:**

> Die SOB ist ein **gut aufgestelltes Unternehmen mit hoher Motivation** (S1-S5),
> dessen **Strategie diese Energie nicht kanalisiert** (H1).
> Die grösste Hebelwirkung liegt nicht in Motivation (schon hoch),
> sondern in **Awareness und Handlungsleitung** -- also darin,
> die Strategie von einem Dokument in ein Steuerungsinstrument zu verwandeln.

Das ist exakt der Auftrag von SOB002.
