## 1. Executive Summary

Das kürzeste und älteste Dokument im SOB-Dokumentenset. Seit **2012** gelten 5 Werte als «Basis des täglichen Handelns». Das Dokument ist ein **Verhaltenskodex** in 5 Einzeilern -- ohne Erläuterung, ohne Beispiele, ohne Operationalisierung.

**Kernbefund:** Die 5 Werte sind **verhaltensökonomisch solide** -- sie decken die wichtigsten Dimensionen sozialer Zusammenarbeit ab. Aber sie sind seit **14 Jahren unverändert** und haben keine sichtbare Verbindung zur aktuellen Strategie, keine Messung und keine Konsequenzen. Der Say-Do Gap von 9 Punkten (DOC-004: 78 «Werte sind klar» vs. 69 «Werte werden gelebt») zeigt: Die Werte werden **gekannt, aber nicht systematisch gelebt**.

**Strategische Relevanz:** Hoch für FehrAdvice. Die Werte sind der «kulturelle Anker» -- sie müssen nicht neu erfunden, sondern **aktiviert, operationalisiert und mit der Strategie verknüpft** werden.

## 2. Die 5 SOB-Werte (Volltext)

| # | Wert | Definition (wörtlich) |
|---|------|----------------------|
| W1 | **Fairness** | «Wir handeln fair und konsequent.» |
| W2 | **Verlässlichkeit** | «Wir sind zuverlässig.» |
| W3 | **Wertschätzung** | «Wir begegnen einander mit Respekt und Wertschätzung.» |
| W4 | **Transparenz** | «Wir kommunizieren offen und ehrlich.» |
| W5 | **Lernbereitschaft** | «Wir sind kritikfähig und lernbereit.» |

## 3. Verhaltensökonomische Analyse der 5 Werte

### 3.1 Mapping auf EBF-Dimensionen

```
┌─────────────────────────────────────────────────────────────────────────┐
│  SOB-WERTE → EBF-DIMENSIONEN                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  W1 FAIRNESS        → Inequity Aversion (Fehr & Schmidt 1999)          │
│     "fair und konsequent"  → Verfahrensgerechtigkeit + Ergebnis        │
│     EBF: U_S (Social Utility), α/β Parameter                          │
│                                                                         │
│  W2 VERLÄSSLICHKEIT → Trust & Reciprocity (Fehr, Gächter 2000)        │
│     "zuverlässig"    → Erwartungserfüllung, Konsistenz                 │
│     EBF: Ψ_S (Soziale Normen), Vertrauen als Kooperationsbasis        │
│                                                                         │
│  W3 WERTSCHÄTZUNG   → Social Recognition (Benabou & Tirole 2003)      │
│     "Respekt"        → Anerkennung intrinsischer Motivation            │
│     EBF: U_IDN (Identity Utility), Warm Glow                          │
│                                                                         │
│  W4 TRANSPARENZ     → Information Asymmetry (Akerlof 1970)            │
│     "offen und ehrlich" → Reduktion von Unsicherheit                   │
│     EBF: AWARE (AU), Informationszugang als Kontextfaktor             │
│                                                                         │
│  W5 LERNBEREITSCHAFT → Growth Mindset (Dweck 2006)                    │
│     "kritikfähig und lernbereit" → Offenheit für Feedback             │
│     EBF: STAGE (AW), Behavioral Change Capacity                       │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 3.2 Abdeckung der Kooperationsdimensionen

| Kooperationsdimension | Abgedeckt durch | Lücke? |
|-----------------------|-----------------|--------|
| **Gerechtigkeit** | W1 (Fairness) | ✅ Abgedeckt |
| **Vertrauen** | W2 (Verlässlichkeit) | ✅ Abgedeckt |
| **Anerkennung** | W3 (Wertschätzung) | ✅ Abgedeckt |
| **Information** | W4 (Transparenz) | ✅ Abgedeckt |
| **Anpassungsfähigkeit** | W5 (Lernbereitschaft) | ✅ Abgedeckt |
| **Innovation** | Nicht direkt | ⚠️ Implizit in W5 |
| **Verantwortung** | Nicht direkt | ❌ Fehlt |
| **Mut / Entscheidungsfreude** | Nicht direkt | ❌ Fehlt |
| **Zusammenarbeit** | Nicht direkt | ❌ Fehlt |

**Fazit:** Die 5 Werte decken die Basis-Kooperationsdimensionen ab, aber **keine handlungsorientierten Werte** (Innovation, Verantwortung, Mut, Zusammenarbeit). Dies passt zum Befund aus DOC-008: Die Strategie hat kein «HOW».

### 3.3 Stärken der Werte-Architektur

| Stärke | Erklärung |
|--------|-----------|
| **Einfachheit** | 5 Werte, je 1 Satz -- einfach merkbar |
| **Konsistenz** | Seit 2012 unverändert → Stabilität |
| **Universalität** | Gelten für alle Ebenen und Funktionen |
| **Innerer Zusammenhang** | Bilden ein logisches System (s.u.) |
| **Schweizer DNA** | Fairness + Verlässlichkeit = typisch CH |

### 3.4 Das innere System der 5 Werte

```
                    W1 FAIRNESS
                   (Spielregeln)
                   /           \
                  /             \
    W2 VERLÄSSLICHKEIT    W3 WERTSCHÄTZUNG
    (Erwartungen           (Beziehungen
     erfüllen)              pflegen)
                  \             /
                   \           /
               W4 TRANSPARENZ
              (Information teilen)
                       |
                       |
              W5 LERNBEREITSCHAFT
              (Sich weiterentwickeln)

  → Die Werte bauen aufeinander auf:
    Fairness braucht Verlässlichkeit und Wertschätzung.
    Beides braucht Transparenz.
    Und Transparenz ermöglicht Lernbereitschaft.
```

### 3.5 Schwächen der Werte-Architektur

```
┌─────────────────────────────────────────────────────────────────────────┐
│  KRITISCHE SCHWÄCHEN DER WERTE (aus EBF-Sicht)                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  S1: KEINE OPERATIONALISIERUNG                                         │
│  "Wir handeln fair und konsequent" -- aber was heisst das?            │
│  • In Gehaltsverhandlungen?                                            │
│  • Bei Beförderungen?                                                  │
│  • Bei Konflikten zwischen GBs?                                        │

│                                                                         │
│  S2: KEINE MESSUNG                                                     │

│  → Werte ohne Messung sind Absichtserklärungen                        │
│                                                                         │
│  S3: KEIN BEZUG ZUR STRATEGIE                                         │

│  → Werte existieren parallel zur Strategie, nicht integriert          │
│                                                                         │
│  S4: KEINE FÜHRUNGSSPEZIFIK                                            │
│  Gleiche 5 Werte für MA und Führungskräfte.                            │

│                                                                         │
│  S5: 14 JAHRE ALT                                                      │
│  Seit 2012 unverändert. Das kann Stärke sein (Stabilität)              │

│  → Werte wurden nie an veränderte Realität angepasst                  │
│                                                                         │
│  S6: KEINE HANDLUNGSWERTE                                             │
│  Alle 5 Werte beschreiben WIE man miteinander umgeht.                 │
│  Keiner beschreibt WAS man tut oder WOHIN man will.                   │
│  Fehlend: Innovation, Mut, Verantwortung, Zusammenarbeit              │

│                                                                         │
```

## 4. Say-Do Gap: Die zentrale Herausforderung

### 4.1 Daten aus MA-Umfrage (DOC-004)

| Frage | Score | Interpretation |
|-------|-------|----------------|
| «Ich kenne die Werte der SOB» | ~78 | Werte sind BEKANNT |
| «Die Werte werden im Alltag gelebt» | ~69 | Werte werden NICHT gelebt |
| **Gap** | **9 Punkte** | **Wissen ≠ Handeln** |

### 4.2 Verhaltensökonomische Erklärung des Say-Do Gaps

```
┌─────────────────────────────────────────────────────────────────────────┐
│  WARUM DER SAY-DO GAP EXISTIERT                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  1. INTENTION-ACTION GAP (Sheeran & Webb 2016)                         │
│     Werte kennen ≠ Werte leben. Ohne konkrete Situationsbezüge        │
│     bleiben Werte abstrakte Absichten.                                 │
│                                                                         │
│  2. SYSTEM 1 vs. SYSTEM 2 (Kahneman 2011)                             │
│     Werte sind System-2-Wissen (bewusst, reflektiert).                 │
│     Alltag ist System-1-Verhalten (automatisch, schnell).              │
│     → Werte werden bei Stress/Zeitdruck "vergessen"                    │
│                                                                         │
│  3. FEHLENDE CHOICE ARCHITECTURE                                       │
│     Kein Default, kein Nudge, kein Feedback-System, das               │
│     wertekonformes Verhalten erleichtert.                              │
│     → Werte brauchen "Infrastruktur", nicht nur Plakate               │
│                                                                         │
│  4. FEHLENDE SOZIALE NORMEN                                            │
│     Wenn Führungskräfte die Werte nicht vorleben,                      │
│     gibt es keine deskriptive Norm ("so macht man das hier").          │
│     → DOC-004: Führungsverhalten Score 67                              │
│                                                                         │
│  5. KEINE KONSEQUENZEN                                                 │
│     Werte-Verletzungen haben keine sichtbaren Konsequenzen.            │
│     Werte-Einhaltung wird nicht anerkannt.                             │
│     → Kein Reinforcement, kein Learning                               │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 5. Implikationen für FehrAdvice

### 5.1 Kernempfehlung: Werte AKTIVIEREN, nicht neu erfinden

```
┌─────────────────────────────────────────────────────────────────────────┐
│  EMPFEHLUNG: WERTE-AKTIVIERUNG IN 4 SCHRITTEN                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  SCHRITT 1: OPERATIONALISIEREN                                         │
│  Für jeden Wert 3-5 konkrete Verhaltensbeispiele definieren:           │
│  "Fairness heisst bei der SOB, dass..."                                │
│  • Entscheidungen transparent begründet werden                         │
│  • Bei Meinungsverschiedenheiten alle gehört werden                    │
│  • Fehler als Lernchance, nicht als Schuld behandelt werden            │
│                                                                         │
│  SCHRITT 2: MESSEN                                                     │
│  Pro Wert 2-3 messbare Indikatoren:                                    │
│  W1 Fairness → "Ich werde fair behandelt" (Score)                      │
│  W4 Transparenz → "Ich werde rechtzeitig informiert" (Score)           │
│  W5 Lernbereitschaft → "Feedback wird umgesetzt" (Score)               │
│                                                                         │
│  SCHRITT 3: NUDGEN                                                     │
│  Choice Architecture für wertekonformes Verhalten:                     │
│  • Meeting-Defaults: "Check-in mit Werte-Bezug"                        │
│  • Feedback-Rituale: Peer-Recognition für Werte-Verhalten              │
│  • Entscheidungs-Templates: "Welcher Wert leitet diese Entscheidung?"  │
│                                                                         │
│  SCHRITT 4: MIT STRATEGIE VERKNÜPFEN                                   │
│  Jeder strategische Schwerpunkt hat einen Werte-Bezug:                 │
│  "Digitalisierung → Lernbereitschaft (W5)"                             │
│  "Zusammenarbeit → Transparenz (W4) + Wertschätzung (W3)"             │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### 5.2 Optionale Erweiterung: Handlungswerte ergänzen

| Bestehend (relational) | Ergänzung (aktiv) | Begründung |
|------------------------|-------------------|------------|
| W1 Fairness | + **Verantwortung** | «Wir übernehmen Verantwortung für unser Handeln» |
| W2 Verlässlichkeit | + **Mut** | «Wir wagen Neues und lernen aus Fehlern» |
| W3 Wertschätzung | (bleibt) | |
| W4 Transparenz | + **Zusammenarbeit** | «Wir arbeiten über Grenzen hinweg zusammen» |
| W5 Lernbereitschaft | + **Innovation** | «Wir suchen aktiv nach besseren Lösungen» |

**Achtung:** Neue Werte nur hinzufügen, wenn die bestehenden 5 zuerst aktiviert sind. Sonst entsteht «Werte-Inflation» ohne Wirkung.
