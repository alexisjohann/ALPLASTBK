## Format-Spezifikationen

| Eigenschaft | Wert |
|-------------|------|
| **Format** | 16:9 (1920 x 1080 px) |
| **Folienanzahl** | 10-15 Folien (Standard: 12) |
| **Farbschema** | SPÖ Rot (#E2001A), Schwarz, Weiss, Hellgrau |
| **Schriftart** | Sans-Serif (z.B. Roboto, Open Sans) |
| **Dateiformate** | PPTX, PDF, Google Slides |

---

## Standard-Struktur (12 Folien)

```
┌─────────────────────────────────────────────────────────────────────────┐
│  FOLIE 1: TITELFOLIE                                                    │
│  FOLIE 2: MANAGEMENT SUMMARY (SCR)                                      │
│  FOLIE 3: STRATEGISCHER RAHMEN (Alt → Neu)                              │
│  FOLIE 4: SÄULEN-ÜBERSICHT                                              │
│  FOLIE 5-8: SÄULEN-DETAILS (je 1 Folie)                                 │
│  FOLIE 9: COUNTER-NARRATIV (Gegner-Antwort)                             │
│  FOLIE 10: Q&A (Kritische Fragen)                                       │
│  FOLIE 11: TIMELINE / ROADMAP                                           │
│  FOLIE 12: CHEAT SHEET (Key Messages)                                   │
└─────────────────────────────────────────────────────────────────────────┘
```

## Die 12 Folien im Detail

### FOLIE 1: Titelfolie

**Elemente:**
- Haupttitel: `[Thema]: Das [Gesetz/Initiative] ([Abkürzung])`
- Untertitel: `Strategisches Briefing zur Kommunikationsoffensive.`
- Visual: Metapher für Transformation (z.B. Chaos → Ordnung)
- Status-Badge: `Status: Strategisches Briefing`

**Beispiel SOG:**
```
Ordnung im digitalen Raum:
Das Social Media Ordnungs-Gesetz (SOG)

Strategisches Briefing zur Kommunikationsoffensive.

[Visual: Punkte von chaotisch zu geordnet]
```

---

### FOLIE 2: Management Summary (SCR-Struktur)

**Elemente:**
| Element | Icon | Beschreibung |
|---------|------|--------------|
| **SITUATION** | 📱 ? | Ausgangslage (1-2 Sätze) |
| **KOMPLIKATION** | ⚠️ | Problem/Herausforderung (1-2 Sätze) |
| **LÖSUNG** | ⚖️ | Unsere Antwort (1-2 Sätze) |

**Zitat-Box am unteren Rand:**
```
„[Kernbotschaft in einem Satz]"
```

**Beispiel SOG:**
```
SITUATION          KOMPLIKATION           LÖSUNG
Der digitale       Dies gefährdet         Das SOG. Kein Verbot,
Raum ist im        Kinder und die         sondern faire
Vergleich zu       Demokratie durch       Spielregeln, wie sie
analogen Medien    Desinformation.        offline längst gelten.
unreguliert.

„Diese Regierung schafft Ordnung. Wer Reichweite hat, trägt Verantwortung."
```

### FOLIE 4: Säulen-Übersicht

│   ICON   │  │   ICON   │  │   ICON   │  │   ICON   │
│          │  │          │  │          │  │          │
│  Titel   │  │  Titel   │  │  Titel   │  │  Titel   │
│          │  │          │  │          │  │          │
│ Untertit │  │ Untertit │  │ Untertit │  │ Untertit │
└──────────┘  └──────────┘  └──────────┘  └──────────┘

**Beispiel SOG (4 Säulen):**

| 3 | 📖🔊 | Medienkompetenz | Demokratie braucht mündige Bürger:innen |
| 4 | 👁️ | Transparenz | Gleiche Spielregeln für alle Medien |

### FOLIEN 5-8: Säulen-Details (je 1 Folie pro Säule)

**Standard-Layout pro Säule:**

```
┌─────────────────────────────────────────────────────────────────────────┐
│  Säule N: [Titel der Säule]                                             │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ┌─────────────────────┐  ┌─────────────────┐  ┌─────────────────────┐ │
│  │                     │  │                 │  │                     │ │
│  │  ZITAT / BOTSCHAFT  │  │  VISUAL/ICONS   │  │  STRATEGISCHER      │ │
│  │  (gross, fett)      │  │                 │  │  STANDPUNKT         │ │
│  │                     │  │                 │  │  (grau Box)         │ │
│  │                     │  │                 │  │                     │ │
│  └─────────────────────┘  └─────────────────┘  └─────────────────────┘ │
│                                                                         │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  Der Mechanismus / Kontext          │  Das Framing / Ziel       │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Beispiel Säule 1 (Altersschutz):**
- Zitat: „Unter 15 Jahren brauchen Kinder Schutz. Das ist konsequenter Jugendschutz – so wie bei Filmen, so wie bei Alkohol."
- Visual: 15+ = Kein Alkohol = Smartphone mit Schutzschild
- Strategischer Standpunkt: „Wenn Europa nicht schnell genug ist, gehen wir national voran."

**Beispiel Säule 2 (Plattform-Haftung):**
- Zitat: „Wer Reichweite hat, trägt Verantwortung. Wer verdient, haftet."
- Visual: Waage mit Gewinn vs. Haftung (DSA Rules)
- Der Mechanismus: DSA bis 6% Jahresumsatz
- Das Framing: Gleiche Spielregeln = Fairness

### FOLIE 9: Counter-Narrativ (Gegner-Antwort)

**Layout:**

```
┌─────────────────────────────────────────────────────────────────────────┐
│  Counter-Narrativ: Die „[Angriff]"-Lüge der [Gegner].                   │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ┌──────────────────┐          ┌────────────────────────────────────┐  │
│  │                  │          │  Die strategische Antwort          │  │
│  │  [Gegner] ruft:  │   ───►   │                                    │  │
│  │  [Angriff]!      │          │  ■ Punkt 1 (Glaubwürdigkeit)       │  │
│  │                  │          │  ■ Punkt 2 (Motiv entlarven)       │  │
│  └──────────────────┘          │  ■ Punkt 3 (Eigenes Argument)      │  │
│                                └────────────────────────────────────┘  │
│                                                                         │
│  ┌─────────────────────────────────────────────────────────────────┐   │
│  │  [CLOSER IN ROT]                                                 │   │
│  └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Beispiel SOG (FPÖ-Zensur):**
- Gegner: „Die FPÖ ruft: Zensur!"
- Punkt 1: „Unter FPÖ-Regierungsbeteiligung ist nichts passiert. Null."
- Punkt 2: „Die meinen: Keine Regeln für ihre Freunde in den Tech-Konzernen."
- Punkt 3: „Im Fernsehen gelten Regeln. Warum soll TikTok keine haben?"
- Closer: **„Das ist keine Zensur. Das ist Ordnung."**

## Qualitätskriterien für Präsentationen

- [ ] Titelfolie mit Kernthema und Transformation-Visual
- [ ] Management Summary folgt SCR (Situation-Komplikation-Lösung)
- [ ] Strategischer Rahmen zeigt Alt → Neu Transformation
- [ ] Jede Säule hat eigene Detail-Folie
- [ ] Counter-Narrativ mit 3-Punkte-Gegenargument
- [ ] Q&A mit 2-3 häufigsten kritischen Fragen
- [ ] Timeline zeigt Handlungsfähigkeit
- [ ] Cheat Sheet ist standalone nutzbar
- [ ] Max. 50 Wörter pro Folie (Lesbarkeit)
- [ ] Jede Folie hat 1 klare Botschaft
