## 1. ZENTRALES ZIEL: Inflationserwartungen verankern — ohne soziale Blindheit

Inflation wird stark von Erwartungen getrieben:
- Arbeitnehmer:innen → Lohnforderungen
- Unternehmen → Preissetzung
- Haushalte → Konsumvorziehen
- Tarifpartner → Indexierungen

**Wenn Politik signalisiert: «Preise werden weiter explodieren» → passiert genau das.**
**Wenn Politik glaubwürdig signalisiert: «Wir begrenzen Preisauftrieb sozial ausgewogen» → sinkt der Druck.**

### Evidenz

```
┌─────────────────────────────────────────────────────────────────────────┐
│  📊 EVIDENZ-BOX: Erwartungskanal                                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Binetti, Nuzzi & Stantcheva (2024):                                   │
│  → Inflation ist das #1-Thema der Wähler:innen (vor Arbeitslosigkeit)  │
│  → Subjektiv wahrgenommene Inflation >> tatsächliche Inflation          │
│  → Falsche Inflationswahrnehmung → höhere populistische Anfälligkeit   │
│  → KAUSALEFFEKT: Inflationssorge → Kontrollverlust → Sündenbocksuche   │
│                                                                         │
│  Romer & Romer (2024):                                                 │
│  → Historische Analyse 1960-2022: Anti-Inflationspolitik WIRKT          │
│  → Aber: kurzfristige Kosten → langfristiger Nutzen                    │
│  → Glaubwürdige Ankündigung = halber Sieg (Erwartungskanal)            │
│                                                                         │
│  Dao, Gourinchas, Leigh & Mishra (2024):                               │
│  → 2021-2023: Disinflation OHNE Rezession möglich                      │
│  → Schlüssel: Erwartungen verankert halten                              │
│  → Wenn Erwartungen entankern → Kosten vervielfachen sich              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Implikation:** Jede öffentliche Äusserung zur Inflation IST eine Intervention in den Erwartungskanal. Nicht ob man kommuniziert ist die Frage — sondern wie.

## 2. DIE VIER KOMMUNIKATIONSPRINZIPIEN MIT REALEM EINFLUSS

### Prinzip 1: Klar sagen, dass hohe Inflation nicht akzeptiert wird

(Auch wenn man sozial handelt.)

**Wirksam:**
> «Hohe Inflation schadet besonders Arbeitnehmer:innen — unser Ziel ist Preisstabilität UND soziale Sicherheit.»

**Unwirksam:**
> «Inflation ist halt da, Hauptsache wir gleichen alles aus.»

**Warum wichtig?** Wenn Politik Inflation relativiert, steigt die Erwartung, dass sie dauerhaft hoch bleibt → Lohn-Preis-Spirale.

```
┌─────────────────────────────────────────────────────────────────────────┐
│  📊 EVIDENZ-BOX: Warum «nicht akzeptieren» wirkt                        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Pallotti et al. (2024) – JME, ECB/Princeton:                          │
│  → Inflation kostet 3% (FR/ES) bis 9% (IT) des Konsums                │
│  → Rentner:innen verlieren bis -14% (Nominalpositionen!)               │
│  → Untere Einkommensquintile am stärksten betroffen                    │
│  → ABER: ~25% der Haushalte GEWINNEN durch Inflation                   │
│     (Hypothekenbesitzer:innen, Staat)                                  │
│                                                                         │
│  → KOMMUNIKATIONS-IMPLIKATION:                                         │
│    «Inflation ist nicht neutral — sie verteilt um.                      │
│     Und sie verteilt VON UNTEN NACH OBEN.»                             │
│    Das ist ein SOZIALDEMOKRATISCHES Argument gegen Inflation.           │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Sozialdemokratischer Killer-Satz:**
> «Inflation ist die unsozialste aller Steuern. Sie trifft jene am härtesten, die am wenigsten haben. Genau deshalb bekämpfen wir sie.»

---

### Prinzip 2: Gezielte Entlastung statt pauschales «Wir zahlen alles»

Sozialdemokratische Kommunikation muss **explizit antizyklisch** sein:

**Gut:**
> «Wir helfen jenen, die es brauchen — ohne zusätzliche Inflation anzuheizen.»

**Schlecht:**
> «Niemand soll irgendetwas spüren.»

**Ökonomischer Effekt:**
- Pauschale Transfers → Nachfrage ↑ → Inflation ↑
- Zielgerichtete Transfers → soziale Stabilität ohne starken Preisdruck

**Das ist nicht neoliberal — das ist makroökonomisch sauber links.**

```
┌─────────────────────────────────────────────────────────────────────────┐
│  📊 EVIDENZ-BOX: Transfers vs. Löhne — was treibt Inflation?           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Köppl-Turyna & Schwarzbauer (2023) — EcoAustria/BMF-Studie:          │
│  → VAR-Modell für Österreich: 1% Lohn → 0,59% Preisanstieg (4Q)      │
│  → Transfers sind WENIGER inflationär als Löhne bei gleichem €1 Mrd.  │
│  → Zentralisierte Lohnverhandlung erzeugt «Jänner-Schock»             │
│    (alle KV-Abschlüsse auf einmal → saisonaler Inflationsschub)       │
│                                                                         │
│  Reiss & Schneider (2024) — OeNB:                                     │
│  → HVPI-Zerlegung: 1% Lohn → NUR 0,27pp HVPI-Anstieg                 │
│  → BIP-Deflator-Analyse übertreibt Lohneffekt um Faktor 2             │
│  → Importpreise = 34,4% der HVPI-Kosten (im BIP-Deflator unsichtbar) │
│                                                                         │
│  → KOMMUNIKATIONS-IMPLIKATION:                                         │
│    «Gezielte Hilfe für Miete, Energie, Lebensmittel ist WENIGER        │
│     inflationär als pauschale Lohnerhöhungen.                          │
│     Wer das einen 'Sozialkompromiss' nennt, hat die Ökonomie           │
│     auf seiner Seite.»                                                 │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Quantifizierte Argumentationshilfe:**

| Massnahme | Inflationseffekt | Quelle |
|-----------|-----------------|--------|
| 1% Lohnerhöhung (Kollektivvertrag) | +0,27pp HVPI (statisch), +0,59% (dynamisch 4Q) | Reiss & Schneider; Köppl-Turyna |
| MwSt-Halbierung Grundnahrungsmittel | -0,20pp HVPI | IHS-Schätzung 2026 |
| Mietpreisbremse | -0,13pp HVPI | IHS-Schätzung 2026 |
| Senkung Elektrizitätsabgabe | -0,12 bis -0,16pp HVPI | IHS-Schätzung 2026 |
| Verbund/EVN Preissenkungen | -0,11 / -0,05pp HVPI | IHS-Schätzung 2026 |
| **Anti-Shrinkflation-Gesetz** (April 2026) | Indirekt: Preisdisziplin ↑ (Strafen bis 15'000 €) | Regierungsbeschluss Feb. 2026 |
| **Preisradar** (Statistik Austria) | Indirekt: Preistransparenz ↑ → Wettbewerb ↑ | Statistik Austria, 100 Produkte live |

### Prinzip 3: Löhne schützen, aber keine Eskalationslogik befeuern

Heikler Punkt — kommunikativ **entscheidend**.

**Stabilisierend:**
> «Reallöhne sichern wir über mehrere Jahre — nicht durch maximale Einmalabschlüsse.»

**Destabilisierend:**
> «Löhne müssen die Inflation vollständig und sofort ausgleichen.»

**Warum?**
- Sofortiger Vollausgleich → Unternehmen preisen künftige Kosten ein
- Mehrjährige Reallohnpfade → Erwartungsdruck sinkt

**Sozialdemokratische Glaubwürdigkeit entsteht hier durch Verlässlichkeit, nicht Maximalrhetorik.**

```
┌─────────────────────────────────────────────────────────────────────────┐
│  📊 EVIDENZ-BOX: Die Lohn-Preis-Spirale ist KLEINER als gedacht        │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  ZENTRALE ERKENNTNIS (Reiss & Schneider 2024):                         │
│                                                                         │
│  BIP-Deflator-Analyse (FALSCH verwendet):                              │
│  → Löhne = 54,3% der Wertschöpfung                                    │
│  → 1% Lohn → 0,54pp Preisanstieg                                      │
│  → SUGGERIERT: Massive Lohn-Preis-Spirale!                             │
│                                                                         │
│  HVPI-Zerlegung (KORREKT):                                             │
│  → Löhne = NUR 26,9% der HVPI-Kosten                                  │
│  → 1% Lohn → NUR 0,27pp HVPI-Anstieg                                  │
│  → REALITÄT: Spirale ist HALB so gross wie behauptet!                  │
│                                                                         │
│  WARUM DER UNTERSCHIED?                                                │
│  → HVPI enthält 34,4% Importkosten (im BIP-Deflator: 0%)              │
│  → HVPI enthält 18,0% Produktsteuern (im BIP-Deflator: 0%)            │
│  → Wer BIP-Deflator für HVPI-Analyse nutzt, VERDOPPELT den            │
│    Lohneffekt künstlich                                                │
│                                                                         │
│  → KOMMUNIKATIONS-IMPLIKATION:                                         │
│    Wenn Wirtschaftsvertreter:innen mit BIP-basierter Analyse           │
│    «Lohnzurückhaltung» fordern: SPÖ kann auf HVPI-Zerlegung           │
│    verweisen und NACHWEISEN, dass der Lohneffekt übertrieben wird.     │
│                                                                         │
│  Dao et al. (2024):                                                    │
│  → 2021-2023: Disinflation gelang OHNE Reallohnverluste               │
│  → Entscheidend: Erwartungen verankert + Angebotsseite erholt sich     │
│  → «Immaculate disinflation» = Inflationsrückgang ohne Jobverluste     │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Sozialdemokratischer Killer-Satz:**
> «Wer behauptet, faire Löhne treiben die Inflation, nutzt die falsche Statistik. 1 Prozent Lohnerhöhung bedeutet 0,27 Prozentpunkte bei den Verbraucherpreisen — nicht 0,54, wie manche behaupten. Wir schützen Reallöhne UND Preisstabilität.»

### Prinzip 4: Preismacht benennen — aber nicht pauschal «Unternehmen» angreifen

**Wirksam:**
> «Wo Marktmacht missbraucht wird, greifen wir ein. Wettbewerb senkt Preise.»

**Unwirksam:**
> «Die Unternehmen sind schuld an allem.»

**Ökonomisch:**
- Pauschale Schuldzuweisung → Unternehmen sichern Margen vor → Preise ↑
- Konkrete Wettbewerbs- & Regulierungssignale → Preissetzung vorsichtiger

**Linke Politik wirkt hier über Institutionen, nicht über Empörung.**

```
┌─────────────────────────────────────────────────────────────────────────┐
│  📊 EVIDENZ-BOX: «Gierflation» — die differenzierte Wahrheit           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  Reiss & Schneider (2024) — OeNB:                                      │
│  → 2022: Gewinne trugen ~27% zur HVPI-Inflation bei                   │
│  → 2023-2024: Gewinnbeitrag drehte NEGATIV (Margenabsorption!)        │
│  → KUMULIERT seit 2020: Überschussgewinne sind NEGATIV                │
│  → «Gierflation» war temporäres Phänomen, NICHT persistentes          │
│    Profiteering                                                        │
│                                                                         │
│  Lebensmittel spezifisch:                                              │
│  → Bundeswettbewerbsbehörde (BWB): KEINE systematischen               │
│    Gewinnmargen-Erhöhungen im Lebensmittelhandel                      │
│  → Preisanstieg durch Energie + Agrarrohstoffe, NICHT durch           │
│    Gewinnmitnahmen                                                     │
│                                                                         │
│  Köppl-Turyna & Schwarzbauer (2023):                                   │
│  → AT Inflation +0,6pp über EA (strukturell: Gastronomie/Hotels +5pp  │
│    HVPI-Gewicht, zentrale KV-Verhandlung, Energiemix)                  │
│  → NICHT durch systematische Gewinnabschöpfung erklärbar              │
│                                                                         │
│  → KOMMUNIKATIONS-IMPLIKATION:                                         │
│    RICHTIG: «Wo es Marktmachtmissbrauch gab, haben wir eingegriffen.  │
│    Die Bundeswettbewerbsbehörde bestätigt: Bei Lebensmitteln gab es   │
│    keine systematische Abzocke.»                                       │
│                                                                         │
│    FALSCH: «Die Unternehmen haben sich auf Kosten der Menschen         │
│    bereichert!» ← widerspricht der Evidenz (kumulativ NEGATIV)        │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Differenzierte Argumentation nach Sektor:**

| Sektor | Gewinnentwicklung | Kommunikation |
|--------|-------------------|---------------|
| **Energie** | 2022-2023 gestiegen (verzögerte Weitergabe), 2024 gefallen | «Regulierung wirkt — Energiepreise sind um fast 5% gesunken» |
| **Lebensmittel** | Keine systematische Margenerhöhung (BWB bestätigt) | «Faktenbasiert: Der Anstieg kam von Energie und Rohstoffen» |
| **Gastronomie/Hotels** | 2024 teils Margenerholung + Lohndruck | «Wettbewerb stärken, wo Preise nicht durch Kosten begründet sind» |
| **Immobilien** | Überdurchschnittliches Gewinnwachstum | «Mietpreisbremse war nötig — und sie wirkt» |
| **Shrinkflation** | Verdeckte Preiserhöhung (weniger Inhalt, gleicher Preis) | «Ab April strafbar. Bis zu 15'000 Euro. Das ist Ordnung.» |
| **Preistransparenz** | Konsument:innen können Preise nicht vergleichen | «Preisradar: 100 Produkte, jeder kann vergleichen. Transparenz senkt Preise.» |

## 4. DAS IDEALE SOZIALDEMOKRATISCHE NARRATIV

> «Inflation ist kein Naturgesetz.
> Wir bekämpfen sie dort, wo sie entsteht — bei Energie, Mieten und Marktmacht.
> Wir schützen niedrige Einkommen gezielt, statt Preise weiter anzuheizen.
> Und wir sorgen dafür, dass Löhne und Preise wieder planbar werden.»

### Warum das wirkt (EBF-Analyse)

| Satz | Ψ-Mechanismus | Paper-Referenz |
|------|--------------|----------------|
| «kein Naturgesetz» | Kontrollwahrnehmung ↑ → Vertrauenserosion ↓ | Binetti et al. (2024): Kontrollverlust → Populismus |
| «dort, wo sie entsteht» | Kausal korrekte Attribution | Reiss & Schneider: Importpreise + Energie, nicht Löhne |
| «niedrige Einkommen gezielt» | Antizyklisch + gerecht | Köppl-Turyna: Transfers < Löhne bei Inflationswirkung |
| «Löhne und Preise planbar» | Erwartungsanker | Dao et al.: verankerte Erwartungen = Disinflation möglich |

## 5. WIRKUNGSTABELLE: Kommunikationsstil → Inflationseffekt

| Kommunikationsstil | Wirkung | Mechanismus | Evidenz |
|-------------------|---------|-------------|---------|
| **Zielgerichtete Hilfe** | 🔽 dämpfend | Nachfrage gezielt statt breit | Köppl-Turyna: Transfers < Löhne |
| **Preisstabilität klar benennen** | 🔽 stark dämpfend | Erwartungsanker | Dao et al.: verankerte Erwartungen = Disinflation |
| **Institutionelle Lösungen** | 🔽 nachhaltig | Preissetzungsverhalten ändert sich | Reiss & Schneider: BWB + Regulierung wirkt |
| **Pauschale Entlastung** | 🔼 treibend | Nachfrage breit ↑ | Standardmakro: Demand push |
| **Alarmistische Rhetorik** | 🔼 treibend | Erwartungen entankern | Binetti et al.: Inflationssorge → Verhaltensänderung |
| **Pauschale Unternehmensschelte** | 🔼 treibend | Unternehmen sichern Margen vor | Reiss & Schneider: kumuliert NEGATIV |

## 7. BOTTOM LINE

**Sozialdemokratische Politik kann inflationstreu sein, ohne unsozial zu werden.**

Aber nur, wenn die Kommunikation **Sicherheit, Begrenzung und Planbarkeit** ausstrahlt — nicht Dauerkrise und Eskalation.
