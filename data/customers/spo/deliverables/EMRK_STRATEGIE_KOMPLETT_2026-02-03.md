# 2. DIE «ORDNUNG STATT SPALTEN»-STRATEGIE

## 2.1 Der strategische Kontext

Die SPÖ steht vor einer kommunikativen Herausforderung: Die FPÖ dominiert das Thema Migration mit emotionaler Rhetorik. Die Versuchung ist gross, mit gleichen Mitteln zu antworten – härter, lauter, emotionaler. **Das ist eine Falle.**

```
┌─────────────────────────────────────────────────────────────────────────┐
│  DIE STRATEGISCHE ENTSCHEIDUNG                                          │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  OPTION A: Der FPÖ folgen                                               │
│  → Immer schärfer, immer härter                                        │
│  → PROBLEM: Die FPÖ gewinnt das Spiel auf ihrem Terrain                │
│  → ERGEBNIS: Wähler wählen das Original, nicht die Kopie               │
│                                                                         │
│  OPTION B: Ordnung statt Spalten                             ← GEWÄHLT │
│  → Faktenbasiert, konsequent, aber menschlich                          │
│  → VORTEIL: Eigenes Terrain, eigene Spielregeln                        │
│  → ERGEBNIS: Differenzierung + Glaubwürdigkeit                         │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## 2.2 Das Grundprinzip

> **«Ordnung statt Spalten» bedeutet: Konsequent handeln UND menschlich bleiben.**

Die SPÖ kann bei Migration **nicht** glaubwürdig «härter als die FPÖ» sein. Aber sie kann **glaubwürdig** sein bei:

| SPÖ kann glaubwürdig | SPÖ kann NICHT glaubwürdig |
|:---------------------|:---------------------------|
| Faktentreue | «Härter als Kickl» |
| Ordnung mit System | Emotionale Rhetorik |
| Traiskirchen-Erfahrung | Angst-Schürerei |
| Rechtsstaat-Kompetenz | EMRK-Feindschaft |

| **L1 + L2 + L3** | **Überzeugend** | **Ordnung auf allen Ebenen** |

**Mathematisch (K8):**

```
W(L1 + L2 + L3) > W(L1) + W(L2) + W(L3)

Die Wirkung kombinierter Levels ist GRÖSSER als die Summe der Teile.
```

# Appendix D: Der Representativeness Bias

## D.1 Warum «50% Straftäter» nicht funktioniert

**Das Problem:** Wenn wir sagen «50% der Abgeschobenen waren Straftäter», passiert im Kopf der Zuhörer Folgendes:

```
Aussage:         «50% waren Straftäter»
                        ↓
Unbewusste Assoziation:  «Die andere Hälfte war auch verdächtig»
                        ↓
Generalisierung:        «Diese ganze Gruppe ist kriminell»
                        ↓
Ergebnis:               FPÖ-Narrativ bestätigt
```

Das nennt man **Representativeness Bias** (Kahneman & Tversky, 1972).

## D.2 Die Lösung

Statt mit Kriminalität zu **beginnen**, beginnen wir mit **System-Erfolg**:

```
Aussage:         «95% waren EMRK-konform»
                        ↓
Assoziation:     «Das System funktioniert»
                        ↓
Kontext:         «Die Problemfälle sind minimal (10-50/Jahr)»
                        ↓
Ergebnis:        FPÖ-Narrativ zerstört
```

## D.3 Die Regel

| ❌ NICHT | ✅ STATTDESSEN |
|:---------|:---------------|
| «50% Straftäter» | «95% EMRK-konform» |
| «Fast die Hälfte kriminell» | «Grösstenteils EU-Bürger» |
| Mit Kriminalität beginnen | Mit System-Erfolg beginnen |
