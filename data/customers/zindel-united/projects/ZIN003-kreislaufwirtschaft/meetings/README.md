# Meetings - ZIN003 Kreislaufwirtschaft

Ablage fuer Praesentationen und Ergebnisse der Projekt-Meetings.

## Namenskonvention

```
YYYY MM DD [Typ] - [Beschreibung].[ext]
```

**Typen:**
- `Praesentation` - Slide Decks, Vortraege
- `Protokoll` - Meeting-Protokolle, Notizen
- `Ergebnis` - Resultate, Entscheidungen, Deliverables
- `Agenda` - Tagesordnungen

**Beispiele:**
- `2026 02 25 Praesentation - Statusupdate Pilotprojekt.pdf`
- `2026 02 25 Protokoll - Steuerungsausschuss.md`
- `2026 02 25 Ergebnis - Entscheidungsvorlage Kostenverteilung.pdf`
