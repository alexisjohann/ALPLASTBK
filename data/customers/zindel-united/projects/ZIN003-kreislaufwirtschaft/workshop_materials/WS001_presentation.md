```

└─────────────────────────────────────────────────────────────────────────────┘
```

## Drei Zustände

| Zustand | Bedeutung | Aktion |
|---------|-----------|--------|

# Der 5+1 Fragen-Filter

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   FRAGEN 1-5: DÜRFEN WIR?                                                   │
│   ─────────────────────────────────────────────────────────────────────────│
│   1. STRATEGIE       Bindet das die Firma langfristig?                      │
│   2. RISIKO          Ist das Risiko unbegrenzt oder schwer abschätzbar?     │
│   3. ZIELKONFLIKT    Gibt es einen Konflikt, den niemand im Raum            │
│                      entscheiden darf?                                      │
│   4. RAHMEN          Budget, Zeitplan oder Richtlinien überschritten?       │
│   5. VERANTWORTUNG   Ist unklar, wer entscheidet?                           │
│                                                                             │
│   → JA bei einer Frage?  →  🔵 ESKALIEREN (via Jörg zu Andreas)             │
│                                                                             │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│   FRAGE 6: WOLLEN WIR JETZT?                                                │
│   ─────────────────────────────────────────────────────────────────────────│
│   "Ist JETZT der richtige Zeitpunkt, um das zu testen?"                     │
│                                                                             │
│   → JA    →  🟢 TESTEN      (Lern-Hypothese formulieren, in 48h starten)   │
│   → NEIN  →  🔴 STOPPEN     (dokumentieren warum nicht jetzt)              │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

**Merke:** Fragen 1-5 = Müssen wir nach oben? | Frage 6 = Machen wir's jetzt?

**→ Gleich üben wir das an 3 Situationen aus eurem Pilotprojekt**
