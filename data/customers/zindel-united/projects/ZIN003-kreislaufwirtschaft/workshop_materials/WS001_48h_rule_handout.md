## Die drei Zustände

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   🟢 TESTEN                                                             │
│   ─────────────────────────────────────────────────────────────────── │
│   Entscheidung JA → Schnell entscheiden, schnell lernen                 │
│                                                                         │
│   → Lern-Hypothese formulieren                                          │
│   → Experiment starten (innerhalb 48h!)                                 │
│   → Aus Ergebnis lernen                                                 │
│                                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   🔴 BEWUSST STOPPEN                                                    │
│   ─────────────────────────────────────────────────────────────────── │
│   Bewusste Entscheidung GEGEN Weitermachen                              │
│                                                                         │
│   → Dokumentieren warum                                                 │
│   → Ressourcen freigeben                                                │
│   → Kein schlechtes Gewissen - das ist auch eine Entscheidung!          │
│                                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   🔵 ESKALIEREN                                                         │
│   ─────────────────────────────────────────────────────────────────── │
│   Eskalationsfilter greift → UL-Meeting einbeziehen                     │
│                                                                         │
│   → Klarer Eskalationspfad                                              │
│   → UL-Meeting entscheidet                                              │
│   → NICHT endlos analysieren - eskalieren!                              │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

## Der 5-Fragen-Eskalationsfilter

**Gehe diese 5 Fragen durch. Bei JEDER Frage mit JA → sofort 🔵 ESKALIEREN**

| # | Frage | ✓ |
|---|-------|---|
| 1 | **Strategische Richtungsentscheidung?** | ☐ |
|   | *Beispiel JA: "Wir steigen aus Kreislaufwirtschaft aus"* | |
|   | *Beispiel NEIN: "RC-Beton im Pilot testen"* | |
| 2 | **Nicht begrenzbares Risiko?** | ☐ |
|   | *Beispiel JA: "Liefervertrag ohne Ausstiegsklausel"* | |
|   | *Beispiel NEIN: "Pilotprojekt mit definiertem Budget"* | |
| 3 | **Legitimer Zielkonflikt, den niemand im Raum entscheiden darf?** | ☐ |
|   | *Beispiel JA: "Bau vs. Logistik – keiner hat Mandat über beide"* | |
|   | *Beispiel NEIN: "Alle Betroffenen sind im Raum und einverstanden"* | |
| 4 | **Rahmen überschritten (Budget, Zeitplan, Richtlinien)?** | ☐ |
|   | *Beispiel JA: "Budget +50%" oder "Beschaffung >10k ohne GL-Unterschrift"* | |
|   | *Beispiel NEIN: "Innerhalb Pilotbudget und Freigabegrenzen"* | |
| 5 | **Verantwortung blockiert?** | ☐ |
|   | *Beispiel JA: "Niemand will unterschreiben"* | |
|   | *Beispiel NEIN: "Pilotleiter hat Mandat"* | |

## Lern-Hypothese (bei 🟢 TESTEN)

Wenn du 🟢 TESTEN entscheidest, formuliere eine Lern-Hypothese:

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   "Wir glauben, dass _____________________________________________     │
│                                                                         │
│    zu ____________________________________________________________     │
│                                                                         │
│    führt.                                                               │
│                                                                         │
│    Wir messen ____________________________________________________     │
│                                                                         │
│    über __________________________________________________________"    │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Beispiel:**
> "Wir glauben, dass die Bauteilbörse bei einem skeptischen Bauherrn
> zu einer positiven Wahrnehmung und Folgeaufträgen führt.
> Wir messen: Kundenfeedback, Mehrkosten, Qualität nach 3 Monaten."

## Die Falle vermeiden

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │
│   ⚠️  DIE HÄUFIGSTE FALLE:                                              │
│                                                                         │
│   🟢 TESTEN Entscheidungen werden behandelt wie 🔵 ESKALIEREN.          │
│                                                                         │
│   "Wir müssen das noch genauer anschauen..."                            │
│   "Ich warte auf mehr Informationen..."                                 │
│   "Das sollten wir noch mit XY besprechen..."                           │
│                                                                         │
│   → Das kostet Zeit, Energie und CHANCEN.                               │
│                                                                         │
│   💡  MERKE:                                                            │
│                                                                         │
│   Bei 🟢 TESTEN Entscheidungen ist NICHT-ENTSCHEIDEN                    │
│   TEURER als eine "falsche" Entscheidung.                               │
│                                                                         │
│   Denn: Du kannst es ja korrigieren.                                    │
│   Aber die verlorene Zeit kommt nicht zurück.                           │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
