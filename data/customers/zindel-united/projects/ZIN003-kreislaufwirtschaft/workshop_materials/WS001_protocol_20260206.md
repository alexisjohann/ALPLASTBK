**Verhaltensökonomische Einordnung:**
- **ARGE als psychologisches Containment:** Die ARGE-Struktur (Arbeitsgemeinschaft) funktioniert als Verantwortungs-Container. Probleme werden von «unlösbar» zu «zugeordnet» transformiert - nicht durch inhaltliche Lösung, sondern durch strukturelle Zuordnung.
- **Relevanz für Diagnose v2.0:** Bestätigt die Kerndiagnose - das Problem ist NICHT Wissen/Motivation, sondern die Entscheidungsstruktur. Die ARGE IST bereits eine Entscheidungsstruktur, die das Team nutzt. Die 48h-Regel muss sich daran andocken.
- **Implikation für Block 4 (Pfosten):** ARGE-Logik als Brücke nutzen - Pfosten können ähnlich wie ARGEs funktionieren (Zuordnung → Verantwortung → Handlung).

```
┌─────────────────────────────────────────────────────────────────────────┐
│                                                                         │

│                                                                         │
│  Das Pilotprojekt darf auf KEINEN FALL eine ARGE werden.               │
│                                                                         │
│  ARGE                          │  STRATEGISCHE INITIATIVE               │
│  ─────                         │  ────────────────────────              │
│  Temporäre Zweckgemeinschaft   │  Gut orchestriert                      │
│  Problemzuordnung              │  Profitabel                            │
│  Reaktiv                       │  Kooperativ                            │
│  Projektbezogen (endet)        │  Strategisch (bleibt)                  │
│  Risiko-Splitting              │  Wertschöpfung                         │
│                                │                                        │
│  → Das Team kennt ARGE als Muster. Genau DESHALB besteht              │
│    die Gefahr, dass es in den ARGE-Modus fällt.                        │
│                                                                         │
│  FACILITATOR-AUFGABE:                                                   │
│  Wenn ARGE-Sprache aufkommt → sofort reframen:                         │
│  «Das hier ist keine ARGE. Das ist EURE strategische Initiative.       │
│   ARGEs enden. Das hier soll bleiben und profitabel werden.»           │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

**Verhaltensökonomische Einordnung der Leitplanke:**
- **Status Quo Bias:** ARGE ist das bekannte Muster im Bau. Das Team wird unbewusst dorthin gravitieren, weil es vertraut ist.
- **Framing-Effekt:** «ARGE» framt als temporär, risikoteilend, reaktiv. «Strategische Initiative» framt als dauerhaft, wertschöpfend, proaktiv.
- **Identität (U_IDN):** ARGE = «wir teilen Risiko». Initiative = «wir schaffen Wert». Verstärkt Andreas' CEO-Framing («wir KÖNNEN»).
- **Konkreter Test in Block 4:** Wenn jemand sagt «das schreiben wir auf die ARGE» → Intervention: «Wer ist OWNER? Was ist der BUSINESS CASE? Wann ist der ERSTE TEST?»
