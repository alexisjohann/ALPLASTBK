# Assura Documents

Upload relevant documents for the Assura project here.

## Suggested Document Types

| Type | Description | Example |
|------|-------------|---------|
| **Annual Reports** | Geschäftsberichte | `assura_geschaeftsbericht_2024.pdf` |
| **Presentations** | Meeting-Präsentationen | `assura_kickoff_2026-01.pptx` |
| **Proposals** | Offerten, Angebote | `assura_beatrix_offerte_v1.pdf` |
| **Meeting Notes** | Protokolle | `assura_meeting_2026-01-28.md` |
| **Analysis** | Analysen, Reports | `assura_market_analysis.xlsx` |
| **Contracts** | Verträge | `assura_rahmenvertrag_2026.pdf` |

## Naming Convention

```
assura_<type>_<description>_<date>.<ext>
```

Examples:
- `assura_gb_2024.pdf`
- `assura_meeting_notes_2026-01-28.md`
- `assura_beatrix_proposal_v2.pdf`

## Notes

- Large files (>50MB) should be stored externally and linked
- Confidential documents should be marked with `_confidential` suffix
- Version documents with `_v1`, `_v2` etc.
