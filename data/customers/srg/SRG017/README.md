# SRG017 - Personalbefragung

Projektordner für die SRG-Personalbefragung.

## Ordnerstruktur

```
SRG017/
├── README.md                        ← Diese Datei
├── projekt_personalbefragung.yaml   ← Projekt-Metadaten
├── fragebogen/                      ← Fragebogen-Versionen
├── daten/                           ← Rohdaten & Datensätze (anonymisiert!)
├── analysen/                        ← Statistische Auswertungen
├── reports/                         ← Ergebnisberichte
└── praesentationen/                 ← Präsentationen
```

## Nutzung

1. **Dokumente ablegen** in den passenden Unterordner
2. **projekt_personalbefragung.yaml** mit Metadaten aktualisieren
3. **Keine personenbezogenen Daten** ohne Anonymisierung in `daten/`

## Datenschutz

⚠️ **WICHTIG:** Alle Daten müssen anonymisiert sein bevor sie hier abgelegt werden.
Keine Namen, E-Mails oder andere personenbezogene Daten in diesem Repository.
