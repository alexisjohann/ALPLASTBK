## Abstract

This paper presents an analytical framework for the study of household production
and the allocation of time within the household. Households are modeled as
producing outputs (commodities Z_i) that bundle goods and time. The household
faces both time and budget constraints, which under the model's assumptions
collapse into a single "full income" constraint: B = wT + V. The shadow price
of each commodity depends on both goods prices and the opportunity cost of time.
This framework allows deeper understanding of consumer choice, income and
substitution effects, and predicts substitution between time-intensive and
goods-intensive commodities.

---

## Core Contribution

Foundational paper for household economics. Established:

1. **Time as scarce resource** with opportunity cost (shadow price = wage)
2. **Household production function:** Z_i = f(X_i, T_i)
3. **Full income concept:** B = wT + V (wage x time endowment + non-labor income)
4. **Commodity price:** pi_i = p_X + w*t_i (goods price + time cost)
5. **Two-stage budgeting** under homogeneous separability

## Core Equations

### Utility Function
```
U = U(Z_1, Z_2, ..., Z_I)
```
Where Z_i are "commodities" produced by the household.

### Household Production Function
```
Z_i = f^(i)(X_i, T_i)
```
Each commodity Z_i is produced using market goods X_i and time T_i.

### Budget Constraint (Full Income)
```
B = wT + V
```
Where:
- B = Full income (maximum potential income)
- w = Market wage rate
- T = Total time endowment (e.g., 24 hours/day)
- V = Non-labor income (property income, transfers)

### Time Constraint
```
T = Sum(T_i) + T_w
```
Where:
- T_i = Time spent on commodity i
- T_w = Time spent in market work

### Full Price of Commodity
```
pi_i = p_i + w * t_i
```
Where:
- pi_i = Full (shadow) price of commodity i
- p_i = Market price of goods input
- w = Wage rate (opportunity cost of time)
- t_i = Time input per unit of commodity i

### Combined Constraint
```
Sum(pi_i * Z_i) = B
```
The household maximizes utility subject to the full income constraint.

## Key Theoretical Insights

### 1. Time Has Economic Value
Time is not "free" - it has an opportunity cost equal to the wage rate.
This means that the true cost of activities includes both money and time.

### 2. Substitution Between Time and Goods
As wages rise, time becomes more expensive relative to goods. This predicts:
- Substitution toward goods-intensive activities
- Demand for time-saving technologies
- Changes in household production patterns

### 3. Unified Framework
By combining time and money constraints into full income, the model provides
a unified framework for analyzing:
- Labor supply decisions
- Consumer demand
- Household production choices
- Value of non-market time

### 4. Income vs. Substitution Effects
Changes in wages have two effects:
- **Income effect:** Higher wages increase full income B, allowing more consumption
- **Substitution effect:** Higher wages make time more expensive relative to goods

## Axioms Established (EBF Formalization)

### TA-1: Time Scarcity
```
T = T_market + T_household + T_leisure
```
Time is finite and must be allocated across activities.

### TA-2: Opportunity Cost
```
Shadow price of time = w (marginal wage rate)
```
Time is valued at its best alternative use (market work).

### TA-3: Household Production
```
Z_i = f(X_i, T_i)
```
Commodities are produced by combining goods and time.

### TA-4: Full Income
```
B = wT + V
```
Maximum potential income if all time were sold at wage w.

### TA-5: Commodity Pricing
```
pi_i = p_X + w * t_i
```
Full price includes both monetary and time costs.

## Influence and Extensions

This paper has influenced virtually every area of household economics:

1. **Labor Economics** (Mincer 1962, 1974)
   - Female labor supply growth explained by rising wages

2. **Transportation Economics** (Gronau 1970)
   - Value of time in mode choice

3. **Health Economics** (Grossman 1972)
   - Health as produced commodity

4. **Family Economics** (Leibowitz 1974, Becker 1981)
   - Child quality production, marriage markets

5. **Digital Platform Economics** (Goodman et al. 2026)
   - Time-intensive goods competition, attention markets
