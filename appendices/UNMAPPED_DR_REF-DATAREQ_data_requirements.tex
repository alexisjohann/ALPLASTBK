\textbf{Core Question:} What data does EBF need, what exists, what is missing, and how current is it?

\textbf{Key Concepts:}
\begin{itemize}[nosep]
\item Multi-Level Data (L0-L3): Individual → Group → Organization → Society
\item 10 Ψ-Dimensions: Context variables requiring empirical input
\item Data Availability Status: Have (\checkmark) vs. Need ($\times$)
\item DACH Focus: Switzerland, Austria, Germany specific data

\end{tcolorbox}


\begin{abstract}
This appendix provides the comprehensive data requirements inventory for the Evidence-Based Framework (EBF). It documents all data needs across four hierarchical levels (L0: Individual, L1: Group, L2: Organization, L3: Society) and ten context dimensions ($\Psi$). For each data requirement, we specify: (1) what data is needed, (2) whether it currently exists, (3) primary data sources, (4) last update date, and (5) collection methodology. Special attention is given to DACH-region (Switzerland, Austria, Germany) empirical values with validated sources.
\end{abstract}

\label{sec:dr-overview}

\subsection{The Data Challenge}

The Evidence-Based Framework requires empirical grounding at multiple levels of analysis. Without reliable data, the framework's predictions remain theoretical rather than actionable.

\begin{tcolorbox}[colback=red!5!white, colframe=red!75!black,
    title=The Fundamental Question]
\textbf{What data does EBF need to generate accurate predictions, what do we have, what is missing, and how do we obtain it?}
\end{tcolorbox}
