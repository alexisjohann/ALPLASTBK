## Key Insight

> **Der GPT → Skills Paradigmenwechsel ist BCM's Window of Opportunity.**
>
> GPTs: User muss BCM kennen und aufrufen
> Skills: Agent erkennt behavioral question und invoked BCM automatisch
>
> Dies entspricht genau BCM's WHEN (Ψ) Konzept — Kontext bestimmt Aktivierung.
